Introduction

Only users who belong to a User Group that has the correct permission can access this option.

If you do not have permission, you will not be permitted access to this option.

This program allows users to set up security levels that can then be assigned to users of the system, scheme records and member records in order to correctly control a user’s access to records within Pension Master. Each security level has a corresponding description which you should use to make it obvious to other users what you have intended each security level for.

The following options and screen appear when this program is selected:

  • Add
  • Delete
  • Report
  • Browse
  • Exit

Security Levels Screen

Add A Security Level

Click ‘Add’.

SECURITY LEVEL - Enter a number which will represent your new security level. The number must be between 1 and 99 and must not exist on the Security Level File i.e. you should not already be able to see an entry in the lower half of the screen for this number.

DESCRIPTION - Enter a description for your new security level. This can be up to 50 characters long.

When the security level has been entered click on ‘OK’ to save it to file. You can click on ‘Cancel’ at any stage to exit the add option without saving the entered security level.

Browse For A Security Level

Click on the ‘Browse’ button or the Browse icon on the toolbar to view details in any screen where the data does not fit on the screen.

Once in browse mode, use the arrow keys or the scrollbar on the right of the data to navigate around the data.

Click on ‘OK’ or ‘Cancel’ to leave browse mode.

Delete A Security Level

Click ‘Delete’.

Under the Security Level heading, enter the number of an existing security level that you wish to delete and click on ‘OK’ to continue.

If the security level entered is already being used by a user, scheme or member record, you will not be permitted to delete it. In this case, an error message will be displayed to the screen informing you of this and you will be returned to the main menu. Otherwise, a window will be displayed with the options ‘Yes’ and ‘No’.

If you do not wish to delete the record, click ‘No’ . You will return to the program menu.

If you wish to delete the record click ‘Yes’.

Another window is displayed asking if you are sure.

If you do not wish to delete the record, click ‘No’.

The program returns to the main menu.

If you wish to delete the record, click ‘Yes’. The record is deleted, and the program returns to the main menu.

Printing Security Level Details

The options for printing are described in the document Printing From Pension Master.

Exit Program

Click ‘Exit’. This leaves the program and closes the current browser tab.
