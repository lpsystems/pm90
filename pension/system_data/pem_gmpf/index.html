Introduction

Only users who belong to a User Group that has the correct permission can access this option.

If you do not have permission, you will not be permitted access to this option.

This program allows maintenance of the Section 148 Orders Table. These are the GMP revaluation factors formerly know as Section 21 Orders and apply to UK schemes only.

The following options and screen appear when this program is selected:

  • Add
  • Query
  • Update
  • Report
  • Exit

Section 148 Orders Screen

The deletion of Section 148 Orders is not allowed because one set of orders per year are required for GMP calculations, and the absence of any of these records would cause erroneous results in GMP calculations.

Add A Section 148 Order Record

Click ‘Add’.

YEAR - A value must be the year entered may not already exist on the Section 148 Orders file as this is a key to the file. The year entered should correspond with the year of the 5th April in the tax year e.g. for the tax year 92/93 enter the year 1993 as the 5th April in that tax year is 5/4/93.

DESCRIPTION - Enter a description for the Orders.

ESTIMATED - Indicates whether the displayed year’s Section 148 Orders are just estimates or actually the finalised figures. If you run a calculation in Pension Master that uses estimated Section 148 Orders, a warning will be written to the Quotation Warnings file at that time.

GMP FACTORS - Enter the Section 148 Orders for the year.

If entering factors for the year 1992/93 the Section 148 Order for 1993 will default to 1 and no further factors may be entered.

When the Section 148 Order details have been entered click on ‘OK’ to save the details to file. You can click on ‘Cancel’ at any stage to exit the add option without saving the entered Section 148 Order details.

Query/Search For A Section 148 Order Record

Click ‘Query’. The following pop-up window is displayed:

Section 148 Orders Query Selection Screen

You may select records using the Section 148 Order Year.

Once you have entered your selection criteria, click on ‘OK’ to proceed and retrieve the records from the database or click on ‘Cancel’ to abort the selection and return to the Section 148 Orders main menu.

If any match to the selection criteria is found then the first record will be displayed to the screen and a message displayed to the bottom of the screen.

Record 1 of 2

Viewing Section 148 Order Records

To view the other records selected press N to view the next record, and P to view the previous record. If N is chosen and there are no records after the current record an error message is displayed. If P is chosen and there are no previous records an error message is displayed.

Update Section 148 Order Records

Before Updating, a record must be selected through the query option, and an error message is displayed if a record for update has not been selected.

Click ‘Update’.

The Section 148 Order Year cannot be changed as this is a key to the file. When the changes have been made to the record click on ‘OK’ to save the changes to file, or click on ‘Cancel’ to exit the update option without saving changes.

Printing Section 148 Order Details

Click the ‘Report’ icon.

If records have been queried in the program before the report option was selected the following window is displayed.

Print Options Screen 1

If no records have been selected in the program before the report option was selected the following window is displayed.

Print Options Screen 2

Record Prints the current screen Record.
Query Prints the all the selected record.
Select Allows selection of records for the report using the Section 148 Order Year.
Exit Closes the window and leaves the report option.

If the Select option was chosen and no records are found to match the selection criteria entered, an error message is displayed and new selection criteria should be entered.

If records are found to match the selection criteria the report continues and options are then displayed for printing.

The options for printing are described in the document Printing From Pension Master.

Exit Program

Click ‘Exit’. This leaves the program and closes the current browser tab.
