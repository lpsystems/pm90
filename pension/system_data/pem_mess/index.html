Introduction

Only users who belong to a User Group that has the correct permission can access this option.

If you do not have permission, you will not be permitted access to this option.

This program allows users to set up warning messages that can be used in formulae to produce more meaningful messages in calculations. Users will be allowed to create a message of up to 50 characters which will be allocated a unique message number which can then be referred to when setting up formulae.

The following options and screen appear when this program is selected:

  • Add
  • Report
  • Browse
  • Exit

User Defined Messages Screen

Add A User Defined Message

Click ‘Add’.

MESSAGE NUMBER - This will automatically be created as the next number in sequence. This will be the reference used when selecting the message in formulae.

MESSAGE - Enter the warning message. This can be up to 50 characters long.

When the User Defined Message has been entered click on ‘OK’ to save the message to file. You can click on ‘Cancel’ at any stage to exit the add option without saving the entered User Defined Message.

Once a message has been entered the user will not be able to amend or delete it. This has been set up this way to prevent messages being set up for a particular reason in a formula then being amended to suit another purpose and leaving the original selection with an inappropriate warning message.

Browse For A User Defined Message

Click ‘Browse’ to view details in any screen where the data does not fit on the screen.

Once in browse mode, use the arrow keys or the scrollbar on the right of the data to navigate around the data.

Click on ‘OK’ or ‘Cancel’ to leave browse mode.

Printing User Defined Message Details

The options for printing are described in the document Printing From Pension Master.

Exit Program

Click ‘Exit’. This leaves the program and closes the current browser tab.
