Introduction

Only users who belong to a User Group that has the correct permission can access this option.

If you do not have permission, you will not be permitted access to this option.

This program allows maintenance of the Exchange Rates that are used for multi currency schemes. In order to set up a currency for a fund you need to set up a fund manager code in the codes program. The exchange rates are assigned to the fund and are used during the process of the members contributions. This is to allow for one or many rates across the system. All of the fund codes are displayed when you enter the screen first you need to scroll to the one you want to update before you press update.

The following options and screen appear when this program is selected:

  • Query
  • Update
  • Checked
  • Exit

Exchange Rate Screen

Query

Click ‘Query’.

The following window is displayed:

You may select records by using the Fund Manager Field. You may also use any combination (or even partial combination) of this fields in order to select the required exchange rate record(s).

If you are unsure of the code(s) to Query, click on the lookup icon to see a lookup list of codes.

Once you have entered your selection criteria, click on ‘OK’ to proceed and retrieve the records from the database or click on ‘Cancel’ to abort the selection and return to the Companies main menu.

View

If you have selected more than one manager using the ‘Query’ option, you can navigate between them using the first, previous, next and last icons on the toolbar.

Update Rates

Click ‘Update’.

Now simply enter the new rate for the currency. The available currency will be displayed in a drop-down list. Only rates that have been checked are used in the allocating and unit processing programs.

Check Rates

Click ‘Checked’.

To check a rate simply tick the box beside the rate. The same user that updated the rate can’t check the rate as well.

Exit Program

Click ‘Exit’. This leaves the program and closes the current browser tab.
