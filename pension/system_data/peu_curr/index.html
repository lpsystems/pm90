Introduction

Only users who belong to a User Group that has the correct permission can access this option.

If you do not have permission, you will not be permitted access to this option.

Currency Conversion Screen

Adding Currency Details

To add a new currency, first either type ‘U’ or select the ‘Update’ menu option with the arrow keys and press Return. Next you should select the ‘Add Row’ option from the menu. You will now be able to enter the details of the new currency.

Updating Currency Details

To update a currency, either type ‘U’ or select the ‘Update’ menu option with the arrow keys and press Return. Move to the row that you wish to update and you may now update the details. You will be unable to amend the currency rate if it is set to fixed. You must enter a new date if you change the rate.

Printing Currency Details

Press R, or select the ‘Report’ menu option.

The options for printing are described in the document Printing From Pension Master.

Exit Program

Click ‘Exit’. This leaves the program and closes the current browser tab.
