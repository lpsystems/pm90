Introduction

The data in these screens is used in the SORP calculator.

The following options and screen appear when this program is selected:

  • Update
  • Exit

SORPS Projection Factors - Standard

There is an additional screen 2 that holds the factors for within 5 years to NPD

Update SORPS Projection Factors

Click ‘Update’.

Click on ‘Insert’ to add a blank line ready for input of the new values.

When the factor details have been entered click on ‘OK’ to save the details to file. You can click on ‘Cancel’ at any stage to exit the add option without saving the entered the details.

Browsing SORP Details

Click on the ‘Browse’ button or the Browse icon on the toolbar to view details in any screen where the data does not fit on the screen.

Once in browse mode, use the arrow keys or the scrollbar on the right of the data to navigate around the data.

Click on ‘OK’ or ‘Cancel’ to leave browse mode.

Printing CEP Factor Details

The options for printing are described in the document Printing From Pension Master.

Exit Program

Click ‘Exit’. This leaves the program and closes the current browser tab.
