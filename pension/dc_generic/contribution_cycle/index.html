DC Generic Contribution Cycle

This document aims to set out the regular contribution processing cycle in Pension Master. It does this as a step by step list of activities you need to do to process regular contributions, some of which may not be applicable to all schemes.

To view a printable checklist, click here.

Contribution Request Run

This program calculates expected contributions for a specified group of members. If successful, these are loaded into a Contribution Batch for onward processing.

It is intended that you send the schedule produced by this program to your client to request the contributions that are then due. The client should then ensure they deduct these contributions from the member and send you full details. These will then be reconciled against what was expected in Batch Data Input.

Allocate Contributions

This program allows you to allocate contributions to investment funds as per the members current investment choice.

A report is available to produce a list of any contributions that have not yet been allocated to investment funds for a selected group of members.

Unit Posting

This program allows you to post allocated contributions to units.

A report is available to produce a list of any investment history entries that have not yet had units posted for a selected group of members.

Switches

This program allows you to switch a member (or group of members) investments from one set of investment funds to another. It allows the option of one to many, many to many or many to one with regard to switching from old fund(s) to new fund(s).

This report allows you to produce a list of any switches that have been instigated for the selected group of members, where the money has not yet been dis-invested from the old investment fund.

Asset Transfer

This program allows you to transfer the existing units for a member to the scheme suspense account. It will transfer all units for a member to the scheme suspense account and write a negative unit entry to each of the members funds so that the members total unit holdings will be zero.

Asset Balance

This program allows you to reconcile total units for a fund against those recorded on Pension Master. Any difference between the two values is written to the scheme suspense account.

Fund Balancing

This program allows you to report on members who have had a lifestyle event since the last date they invested contributions. It shows you the current allocation of the members investments, and the proposed new split of investments after re-investing according to the appropriate lifestyle matrix.

Bulk Fund Redirection

This program enables you to add or remove a investment setup details to a selection of members in one step.

Units Listing Report

This program allows you to produce a list of contributions and units for the selected group of members between two user entered dates.
