Introduction

Basic Fund Details Screen

Only users set up as Supervisor status in Users may access this program.

The type of access granted can be view only or full access. For the view only option, you will not be permitted access to any menu option that allows you to add/update/delete data.

If you do not have Supervisor status, you will not be permitted access to this program.

Add

Click ‘Add’.

Basic Fund Details Screen

Enter the details as follows:

FUND CODE - Enter a code for your fund to be saved by. This can be up to 10 characters in length and cannot contain spaces, ‘, “, /, \ or * as they are special characters. Ampersands (&) are allowed.

NAME - Enter a description or name for your fund.

ISIN - Enter the ISIN number of the fund.

ADDRESS - Enter the address of the fund.

POSTCODE - Enter the postcode of the fund.

TEL NO - Enter the telephone number of the fund.

FAX NO - Enter the fax number of the fund.

E-MAIL ADDRESS - Enter the e-mail address of the fund.

CONTACT 1 TITLE, FORENAME, SURNAME - Enter the details of your primary contact at the fund.

CONTACT 2 TITLE, FORENAME, SURNAME - Enter the details of your secondary contact at the fund.

When you have entered these details, click on ‘OK’ to proceed or ‘Cancel’ to return to the main menu.

Fund Restrictions Screen

The following screen is displayed:

Fund Restrictions Screen

Enter the details as follows:

DATE FUND OPENED - Enter the date the fund opened to investments or leave blank. Investments will not be permitted in this fund prior to this date.

DATE FUND CLOSED - Enter the date the fund closed to investments or leave blank. Investments will not be permitted in this fund after to this date.

CAN PROTECTED RIGHTS BE INVESTED IN FUND? - Can Protected Rights contribution types be invested in this fund?

CAN MONIES BE SWITCHED INTO FUND? - Can monies be switched into this fund?

CAN MONIES BE SWITCHED OUT OF FUND? - Can monies be switched out of this fund?

BUY BASIS - Select the unit buy basis for this fund:

  • Bid
  • Mid
  • Offer

SELL BASIS - Select the unit sell basis for this fund:

  • Bid
  • Mid
  • Offer

When you have entered these details, click on ‘OK’ to proceed or ‘Cancel’ to return to the main menu.

The details required to set up your fund have now been entered and you will be returned to the main menu.

Query

Click ‘Query’.

The following window is displayed:

You may select records by using the Fund Code field. You may also use any combination (or even partial combination) of this fields in order to select the required fund record(s).

If you are unsure of the code(s) to Query, click on the lookup icon to see a lookup list of codes.

Once you have entered your selection criteria, click on ‘OK’ to proceed and retrieve the records from the database or click on ‘Cancel’ to abort the selection and return to the Companies main menu.

View

If you have selected more than one fund using the ‘Query’ option, you can navigate between them using the first, previous, next and last icons on the toolbar.

Update

Before Updating, a record must be selected through the Query option, and an error message is displayed if a record for update has not been selected.

Select the screen you wish to update using the ‘Menu’ option, then click ‘Update’.

To update the Basic Fund Details and Fund Restrictions screens, see the ‘Add’ option above. The other four screens that are available are as follows:

Unit Prices Screen

The following screen is displayed:

Unit Prices Screen

Enter the details as follows:

DATE - Enter the date for your unit price. When posting units for members, the process will lookup the appropriate unit price at the date invested for the contribution concerned.

BID PRICE - Enter the unit bid price for the date concerned.

MID PRICE - Enter the unit mid price for the date concerned.

OFFER PRICE - Enter the unit offer price for the date concerned.

When you have entered these details, click on ‘OK’ to proceed or ‘Cancel’ to return to the main menu.

Fund Information Screen

The following screen is displayed:

Fund Information Screen

Enter the details as follows:

CASH FUND? - Tick field if fund is a cash only fund.

DEPOSIT FUND? - Tick field if fund is a deposit fund.

PRSA FUND? - Tick field if fund is classed as PRSA

PRSA TYPE - Select the PRSA Type from the drop-down list.

FUND CURRENCY - Select the Fund Currency from the drop-down list.

FUND NUMBER - Enter the fund number.

FUND MANAGER - Select the Fund Manager from the drop-down list.

PRICE - Enter the price.

TOTAL EXPENSE RATIO - Enter the ratio.

ASSET CLASS - Select the Asset Class from the drop-down list.

NOT ONLINE - Tick field if fund not to be shown on line.

RISK PROFILE - Select the Risk Profile from the drop-down list.

When you have entered these details, click on ‘OK’ to proceed or ‘Cancel’ to return to the main menu.

Asset Mix Screen

The following screen is displayed:

Asset Mix Screen

Enter the details as follows:

Internal Lifestyling? - Enter Yes or No if Fund has Internal Lifestyling.

DATE - Enter the effective date.

EQUITY % - Enter % invested in equity

PROPERTY - Enter % invested in property.

FIXED INTEREST - Enter % invested in fixed interest.

CASH - Enter % of invested in cash.

OTHER - Enter % invested in other.

TOTAL - This field is auto populate and has to be 100.

AMC - Asset Management Charge - Enter the % it can be left blank.

COMMENT - Enter any free text comments on the fund.

When you have entered these details, click on ‘OK’ to proceed or ‘Cancel’ to return to the main menu.

Investment Value Summary

The following screen is displayed:

Investment Value Summary Screen

This is a view only screen that displays a valuation of this fund, split scheme by scheme, based on the latest unit price on the unit prices screen for the appropriate unit sell basis. This is colour coded as follows:

  • Black - Valuation of units based on a unit price within the last 90 days.
  • Blue - Valuation of units based on a unit price older than 90 days. A warning will also be displayed if any unit prices are used that are older than 90 days from today’s date.
  • Red - Unallocated contributions i.e. contributions that have not yet been used to buy units. These are taken at face value e.g. £100 of unallocated contributions has a value of £100.

Delete

Before Deleting, a record must be selected through the Query option, and an error message is displayed if a record for deletion has not been selected.

You cannot delete a fund that has investments in it.

You will be asked if you really wish to delete the fund and if so, the fund and all details pertaining to it will be deleted.

Report

Click the ‘Report’ icon.

If records are found to match the selection criteria the report continues and options are then displayed for printing.

The options for printing are described in the document Printing.

Navigating Between Screens

Click ‘Menu’ to allow you to navigate between the various screens. Alternatively, you can click on the ‘1’, ‘2’, ‘3’ or ‘4’ buttons at the bottom right of each screen.

Exit Program

Click ‘Exit’. This leaves the program and closes the current browser tab.
