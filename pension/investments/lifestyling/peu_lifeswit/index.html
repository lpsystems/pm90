Introduction

This program creates switches for members, as selected by the user, who have a lifestyle event on the switch date entered by the user, where there are units present in the investment funds. The transaction type codes used for lifestyle switches are LIFEIN for switching in and LIFEOUT for switching out. There are two types of Lifestyles either ‘Event’ or ‘Continuous’. An event is actually when the member moves to the next stage of the lifestyle i.e. 2 years to retirement. A continuous type is where the switch can be carry out at all time. The following options and screen appear when this program is selected:

  • Proceed
  • Exit

Selection Criteria Screen

Create Lifestyle Switches

SELECTION CRITERIA - Select the group of members you require to create lifestyle switches for by using the Scheme, Company, Category, Status, Special Code and Member Number fields (see Selection Criteria). When the selection criteria have been entered click ‘OK’ to proceed.

You can click ‘Cancel’ to return to the main menu and abort the lifestyle switches program.

The following screen is displayed:

Switch Date

TYPE - Select either Event or Continuous Fund Rebalancing

BASIS - Select either value or unit.

SWITCH DATE - Enter the switch date for the lifestyle switches you wish to create.

UNIT PRICE EFFECTIVE DATE - Enter the date.

When the switch date has been entered, click on ‘OK’ to proceed. You can click on ‘Cancel’ to return to the main menu and abort the lifestyle switches program. The program will now check through each members data and if any warnings are found then they will be reported out to Excel. Examples of warnings are listed below:

Warnings
No lifestyle matrix entry available for matrix code XXXX and start age nn.
Member has a blank DOB. Cannot calculate age for age related lifestyle event.
Member has a blank TRD. Cannot calculate term for term related lifestyle event.
No lifestyle matrix entry available for matrix code XXXX and term to TRD nn.

The lifestyle switches report that gets produced, will list out any members, from within the selection criteria, who have a lifestyle event on the date entered. The investment setup, for the lifestyle event, will be reported out along with the date that the event will first occur on. All the lifestyle matrices used within the investment setup will also be reported out showing the investment split.

Once the report has been produced, a message will appear to the screen asking the user “Do you wish to set up switches for the selected members who have a lifestyle event occurring on the entered switch date?”. Click on ‘Yes’ to continue and create the switches, or click ‘No’ to return to the main menu.

If any members are found, within the selection criteria, who have a lifestyle event but who don’t have any units at that time to be switched then a report will be produced listing these members.

Also, a switch report will be produced that lists out any members found within the selectin criteria, who have units to switch. The switch report will display all the unit amounts for the LIFEIN and LIFEOUT parts of each member’s switch.

To reduce the size of the LIFEIN and LIFEOUT parts of the switch, the program will just create amounts based on the differences between the two. For example, if a member is changing from 50% into Fund A and 50% into Fund B, to 55% into Fund A and 45% into Fund B, then rather than showing 50% going out of Fund A and 55% coming into Fund A, it’ll just show the 5% coming in.

The following popup window is now displayed to the screen which allows the user to choose which reports they wish to view.

View/Print Report Selection Screen

On clicking ‘OK’ in the ‘View/Print Report Selection’ screen, any reports the user has selected to view will be displayed out to Excel. Once the report(s) have been viewed the user will then be asked ‘Do you wish to create these switches?’. On clicking ‘Yes’, the switches will be created directly onto the Investment History screen for the selected members. Clicking ‘No’, will return the user to the main menu.

Undo a Lifestyle Switch

There is no Undo option available in the Lifestyle Switches program, so if a switch needs to be removed then the user must use the Undo options in the Post Units and Contribution Allocation programs.

Exit Program

Click ‘Exit’. This leaves the program and closes the current browser tab.
