Introduction

This program produces a report out to Microsoft Excel for members, as selected by the user, who have a lifestyle event that’ll occur within the date range entered by the user. The following options and screen appear when this program is selected:

  • Proceed
  • Exit

Selection Criteria Screen

Run A Report

SELECTION CRITERIA - Select the group of members you require to produce this report for using the Scheme, Company, Category, Status, Special Code and Member Number fields (see Selection Criteria). When the selection criteria have been entered click ‘OK’ to proceed.

You can click ‘Cancel’ to return to the main menu and abort the lifestyle event report.

The following screen is displayed:

Start and End Dates

START DATE - Enter the start date for the date range you wish to use.

END DATE - Enter the end date for the date range you wish to use.

When the start and end dates have been entered, click on ‘OK’ to proceed. You can click on ‘Cancel’ to return to the main menu and abort the lifestyle event report. The report will now be created and reported out to Excel.

If any exceptions are found for a member within the selection criteria, then an exceptions report will be produced first, which will also be reported out to Excel. Examples of exceptions are listed below:

Exceptions
No lifestyle matrix entry available for matrix code XXXX and start age nn.
Member has a blank DOB. Cannot calculate age for age related lifestyle event.
Member has a blank TRD. Cannot calculate term for term related lifestyle event.
No lifestyle matrix entry available for matrix code XXXX and term to TRD nn.

The lifestyle event report that gets produced, will list out any members, from within the selection criteria, who have a lifestyle event within the date range entered. The investment setup, for the lifestyle event, will be reported out along with the date that the event will occur on. All the lifestyle matrices used within the investment setup will also be reported out showing the investment split.

Once the report has been produced, a message will appear to the screen stating that the report has been produced successfully.

Exit Program

Click ‘Exit’. This leaves the program and closes the current browser tab.
