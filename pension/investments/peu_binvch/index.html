Introduction

The Bulk Fund Redirection option enables the user to add or remove a investment setup details to a selection of members in one step.

If users wish to redirect member’s accumulated funds to the new investment setup, then they must do so using the Switches program.

The following options and screen appear when this program is selected:

  • Run
  • Undo
  • Exit

Bulk Investment Setup Screen 1

Run

Press R or click on the ‘Run’ menu option.

This option allows a bulk update to be performed. You will be asked if you wish to do a Provisional or an Actual run. A Provisional run does all that an actual run does, but does not write any results to the database.

Enter your selection criteria in the relevant boxes. Once completed click on ‘OK’ and a new screen will be displayed.

Bulk Investment Setup Screen 2

Enter the date from which your investment setup details will become effective in the effective date box.

A window will then appear asking which contribution type you wish to enter first. This lists those contribution types that are applicable for the effective date you have entered. The window is displayed below:

Select the contribution type you want to setup details for and click on ‘OK’. The following window appears:

Enter the split of contributions between investment funds for this contribution type. The ‘% Alloc’ column must total 100%.

If you wish to use a lifestyle matrix for this contribution type, set the fund as ‘LIFESTYLE’ and enter the lifestyle matrix code in the matrix column. If you choose a lifestyle matrix for this contribution type, you can not further split the contribution type between other investment funds i.e. you must either split the whole contribution type between investment funds or specify a lifestyle matrix for the whole contribution type.

When you have finished, click on ‘OK’ to continue. You will now be returned to the contribution type selection window so you can choose another contribution type to setup. After you setup your first contribution type, you will be asked if you wish to apply the same split of investments to all available contribution types. This is a useful shortcut if all contribution types are to be setup the same.

Continue with this process until all contribution types have been setup.

When you have done this, click the ‘Done’ button in the contribution type selection window to continue with and run your bulk investment setup run.

If you have selected an ‘Actual’ run the data you have entered will now be saved to the database unless exceptions are found. If there any exceptions no members will be updated.

Undo

Only users who belong to a User Group that has the correct permission can access this option.

If you do not have permission, you will not be permitted access to this option.

Press U or click on ‘Undo’ menu option.

This option allows a bulk undo to be preformed. This will remove all investment setup details for a given effective date for a selected group of members .

Enter you selection criteria, and click the ‘OK’ button. Now enter the effective date you wish to undo and click the ‘OK’ button.

All investments for members within your selection criteria will now have any investment details deleted.

Exit Program

Click ‘Exit’. This leaves the program and closes the current browser tab.
