Introduction

This program allows you to reconcile total units for a fund against those recorded on Pension Master. Any difference between the two values is written to the scheme suspense account.

This difference may arise, for example, from a build-up of rounding on unit allocations between units held on Pension Master and those allocated by your investment manager. This normally arises if the investment manager holds units and unit prices to different numbers of decimal places than Pension Master.

It is envisaged that as part of your regular contribution cycle, you will check the scheme suspense account for any significant build-up of units and if necessary allocate a unit dividend to re-allocate these suspense units to members accounts.

The following options and screen appear when this program is selected:

  • Run
  • Exit

Asset Balance Screen

Run

Click ‘Run’.

Enter the details as follows:

EFFECTIVE DATE - Enter the effective date for your asset balance. This date will be used to create the entry in the scheme suspense account. You can easily spot entries in the scheme suspense account resulting from an Asset Balance process because they will be for member number 0 and have an appropriate comment against them.

COMMENTS - Enter any comments you may have about this asset balance. These will then be saved to the scheme suspense account against each suspense account entry generated. This field defaults to ‘Asset balance adjustment’.

TOTAL UNITS - This field is automatically populated by Pension Master and will be the total of any member units in this fund (for members in this scheme) plus the total scheme suspense units for this scheme and fund.

REVISED TOTAL UNITS - Enter the total number of units you have in this fund in respect of investments by this scheme. This will usually be found on allocation statements produced by your investment manager. This field defaults to the value held in the TOTAL UNITS field.

When you have entered these details, click on ‘OK’ to proceed or ‘Cancel’ to return to the main menu.

The asset balance process will now run and a record written to the scheme suspense account if you entered REVISED TOTAL UNITS that differ in value from the TOTAL UNITS figure.

A report will be generated which shows details of the units created (if any) on the scheme suspense account.

The program will then return to the main menu.

Exit Program

Click ‘Exit’. This leaves the program and closes the current browser tab.
