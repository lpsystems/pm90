Introduction

This module allows you to switch a member (or group of members) investments from one set of investment funds to another. It allows the option of one to many, many to many or many to one with regard to switching from old fund(s) to new fund(s).

You can also take expenses off switches that a member makes. Further details on how to setup switch expenses can be found here.

Basic Investment Switch Details Screen

Add An Investment Switch

Click ‘Add’.

The following screen is displayed:

Member Selection Screen

This screen allows you to select the member(s) to be included in the switch:

SELECTION CRITERIA - Select the group of members you require to run the Investment Switch for using the Scheme, Company, Category, Status, Special Code and Member fields. When the selection criteria have been entered click on ‘OK’ to proceed or click on ‘Cancel’ to return to the main menu.

The following screen is displayed:

Basic Investment Switch Details Screen

Enter the details as follows:

TYPE - Select the type of switch:

  • Switch - A normal switch where investments are moved from old fund(s) to new fund(s).
  • Sale - A switch out of a fund without the re-investment in new funds.

DATES - Select the date type you wish to use for the switch:

  • Full Switch - A full switch is where investments, that are found on or before the date received, are moved from old fund(s) to new fund(s).
  • Switch Range - A switch range is where investments, that are found between the date range entered, are moved from old fund(s) to new fund(s).

DATE RECEIVED - Enter the date the switch instruction was received. This defaults to today’s date.

This field is renamed ‘DATE TO’ if the user has selected to enter a Switch Range.

DATE FROM - Enter the date that the switch range starts from. This date cannot be after the Date To. This field is only entered if the user has selected to enter a Switch Range.

SWITCH EXPENSES - Specify the type of expenses to apply to this switch:

  • Calculated Expenses - Calculate expenses as specified under Switch Expenses in Scheme File.
  • Free Switch - Do not calculate any expenses for this switch. This is the default option.

SWITCH BASIS - Switch basis can be one of:

  • Monetary amount - Switch units to the value of a given monetary amount per member. Before using this option, see the important note here.
  • Percentage of units - Switch a percentage of the members units. This is the default option and is the one it is envisaged will be used most often.
  • Number of units - Switch a specific number of units per member.

INCLUDE PROT RTS - Specifies whether to include any protected rights contribution types in the switch. The default setting is to include all contribution types.

When you have entered these details, click on ‘OK’ to proceed or ‘Cancel’ to return to the main menu.

The following screen is displayed:

Investment Switch Details Screen

This screen allows you to specify which fund(s) you wish to switch out of and, in the case of a switch, which fund you subsequently wish to switch into.

Enter the switch out details as follows:

FUND - Select the investment fund to switch out of. If you wish to switch out of all funds the member currently invests in, enter ‘ALL’ in this field.

CONT TYPE - Select the contribution type to switch out of. If you wish to switch out of all contribution types, enter ‘ALL’ in this field.

AMOUNT - If you have indicated that the switch is on a ‘Monetary Amount’ basis, enter the amount you wish to realise per member from this fund.

% UNITS - If you have indicated that the switch is on a ‘Percentage Of Units’ basis, enter the percentage of units you wish to realise per member from this fund. This must be between 0% and 100%.

NUM UNITS - If you have indicated that the switch is on a ‘Number Of Units’ basis, enter the number of units you wish to realise per member from this fund.

When you have entered these details, click on ‘OK’ to proceed or ‘Cancel’ to return to the main menu.

Enter the switch in details as follows:

FUND - Select the investment fund(s) to switch into.

PERCENTAGE - Enter the percentage of the total monetary amount realised by the sale to allocate to this fund.

When you have entered these details, click on ‘OK’ to proceed or ‘Cancel’ to return to the main menu.

The switch will now be processed.

Exception Report

If any exceptions have been found then you will be automatically taken into the exceptions report which details the problems found.

Possible exception messages are:

  • There is already a switch from this fund on this date.
  • Unit sell price for fund is zero.Unable to determine number of units to sell for this fund.
  • Member does not have enough investments in this fund to realise the monetary amount requested.
  • Member does not have enough units available to realise the number of units requested.

On closing the exceptions report, you will be returned to the main menu and the switch status flag will be set to ‘Switch failed’. You should then edit your investment switch instructions using the ‘Update’ option and re-run the switch.

Alternativaly, you may delete the switch instruction using the ‘Delete’ option.

Results Reports

If no exceptions have been found then the following window is displayed:

This allows you to view and/or print the following reports:

  • Details Report - Shows details of the switch you have requested as they appear on the screens you have entered for this switch.
  • Sale Report - Shows details of the units to be sold for each fund that you have indicated you wish to switch out of. It is intended that you send this to each old fund(s) as your notification of the number of units to sell in that fund. You could accompany this with a suitable covering letter. The report can be produced in Summary or Detailed format as follows:
    • Summary - one page per fund showing the total units to sell for each contribution type along with a total overall number of units to sell in that fund.
    • Detailed - as for the Summary report except that units to be sold for each member concerned are also printed.

When you have finished with the reports window, click on ‘OK’ to proceed or ‘Cancel’ to return to the main menu.

Switch entries are now created on each individual members record based on the member selection criteria and switch details you entered.

The process of adding a switch is now complete. When you receive notification of units sold and sale value from the old fund(s), use the Sell option to complete the switch process for thus fund.

The program now returns to the main menu.

Query

Click ‘Query’.

This will display the following window which lists all known investment switches:

Select the switch you require then click on ‘OK’.

Click on ‘Cancel;’ to return to the main menu.

Report

Click the ‘Report’ icon.

This displays the Results Reports window as for the ‘Add’ option and allows you to view/print details of the selected investment switch.

Delete

Click ‘Delete’.

This allows you to delete an Investment Switch and all associated Investment History entries for this switch for all members included in the switch.

You will not be allowed to delete a investment switch:

  • Where the entry on the members Investment History relating to the switch in has been posted to units.
  • Where there are any switches after date requested of the switch to be deleted for the a member.

These will be reported as exceptions and the delete of the Investment Switch will be abandoned.

This option will not only delete the switch, but will also delete the corresponding switch details on the selected individual member records and any entries in the members Investment History that are a result of this switch. We thus recommend you are sure you know what you are doing before using this option.

Menu

Click ‘Menu’ to allow you to navigate between the various screens. Alternatively, you can click on the ‘1’ or the ‘2’ buttons at the bottom right of each screen.

Sell

Click ‘Sell’.

The following window is now displayed, where you are prompted to choose whether you wish to enter Unit Prices or Sale Values for the funds you are switching out from:

Instruction Option Screen

If you select to enter Unit Prices then the following window is displayed:

Unit Price Entry Screen

If you select to enter Sale Values then the following window is displayed:

Sale Value Entry Screen

Either window has one line for each of the fund(s) you requested a switch out of. Enter the sale date for those funds you have received proceeds from.

If you have selected to enter Unit Prices, then if the unit price for the sale date entered exists on the system, then the unit price field is automatically filled in with this. If it’s not then the user will be prompted to enter one.

If you have selected to enter Sale Values, then you will need to enter them in the Sale Value field.

When you have entered these details, click on ‘OK’ to proceed or ‘Cancel’ to return to the main menu.

The sale of investments for the entered fund(s) is now processed and you will be presented with the Results Reports window as for the ‘Add’ option.

When you have finished with the reports window, click on ‘OK’ to proceed or ‘Cancel’ to return to the main menu.

If you have entered sale details for all old fund(s), the switch status will be automatically set to ‘Switch finished’. If there are still some fund(s) for which you have not yet received the proceeds, the switch status will be automatically set to ‘Partially finished’ and you should use the ‘Sell’ option again when you receive notification of the proceeds from the remaining fund(s).

Exit Program

Click ‘Exit’. This leaves the program and closes the current browser tab.

Switch Status Codes

The possible switch status codes are:

Description Notes
Switch failed exception report produced
Switch pending awaiting unit prices
Partially finished some funds left to sell
Switch finished fully processed

Switches Based On A Monetary Amount

If a request to switch funds based on a monetary amount is to be processed, it is imperative that the current unit sell price for that fund is ascertained and that is then used to determine the number of units to sell for the member. If the unit sell price drops between this time and the time of actual sale, the administrator should be aware that they will not realise the full amount required and vice-versa if the unit price increases in the meantime.

It may be possible (and indeed desirable) for the scheme to have an agreement with their investment funds to sell units by telephone instruction at a sell price agreed in the phone call. This would then guarantee the correct amount would be realised for the sale.

It is anticipated that this method of sale would be more suited to the insurance company in-house managed scheme where the unit sell prices are known in advance.

Overvew Of The Switch Process

Stage 1 - Input the switch

  1. Use the ‘Add’ option to select the members you wish to apply the switch to.
  2. Establish basic switch details such as date instruction received and any expenses to be taken in respect of the switch.
  3. Establish what fund(s) to sell from and how much to sell. Sale method can be one of:
    • Monetary amount
    • Percentage of units
    • Number of units
  4. Now we must indicate where the proceeds from the switch are to be re-invested. This is done by entering a list of fund code(s) and percentage(s). The total of the percentage column must be 100% across all funds to switch into.
  5. A unit sale report is then produced showing the number of units to sell for each fund concerned, each fund starting on a new page. The report has a summary section at the end and an option to print just the summary section is available. Send this printout (plus perhaps a covering letter) to each fund concerned requesting the sale of the requisite number of units.

Stage 2 - Sell Units From Old Fund(s)

The old fund(s) will inform us of:

  • number of units sold
  • sale date
  • unit price at which the units were sold
  • sale proceeds value

Use the ‘Sell’ option to enter the sale date and value realised for each fund(s) to switch out of.

Pension Master will then calculate each members share of the sale proceeds value. This is the amount each member has available to invest in the new fund(s). Any switch expenses are taken off the amounts (as appropriate) before sending the money to the new investment funds.

Stage 3 - Sell From All Old Fund(s)

Repeat stage 2 above until we have received and input the proceeds from all old fund(s). The switch process as far as Pension Master is concerned is now finished.

Stage 4 - Re-Invest Proceeds

Each members sale proceeds are split up between the new fund(s). A cheque for the total amount for each new fund(s) will be sent to each respective new investment fund(s).

The new investment history entries against the new funds just become normal un-allocated contributions and can be picked up from here in the same way as normal un-allocated contributions.
