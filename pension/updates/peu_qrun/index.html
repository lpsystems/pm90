Introduction

This program allows you to run quotations for a batch (or group) of members.

The following options and screen appear when this program is selected:

  • Run
  • Exit

Batch Quotations Screen

Run A Batch Of Quotations

Click ‘Run’.

SELECTION CRITERIA - Select the group of members you require to run quotations for using the Scheme, Company, Category, Status, Special Code and Member fields (see Selection Criteria). When the selection criteria have been entered click on ‘OK’ to proceed.

You can click on ‘Cancel’ to return to the main menu and abort the quotation run.

The following screen now appears to the screen.

Batch Quotations Screen

UPDATE TYPE - a window will appear listing all the available Quotation Types for this scheme. Move the cursor to the Quotation Type you require using the arrow keys and click on ‘OK’ to select it. The Quotation Type code and description will be displayed to the screen.

CALCULATION DATE - Enter the calculation date to be used in your quotations. This date is the effective date of the quotation. The calculation date field is held for each members quotation (pmqh_calcdate).

QUOTATION DATE - Enter your quotation date. This date is used when extracting factors.

Click on ‘OK’ to proceed with the batch quotation. If you click on ‘Cancel’, the batch quotation process will be abandoned and the user will be returned to the main menu.

If you choose to proceed with the process, then you will be asked ‘Do you wish to run the batch quotations?’. On selecting ‘Yes’, the program will continue to process the quotations.

You will then be prompted to see if you wish to print/view various reports as follows:

  • Warnings report - This report lists any errors or warnings that have been generated in running the quotations e.g. Attempt to divide by zero..
  • Calculation Exceptions report - This report lists those members whose quotation failed followed by an appropriate reason e.g. ‘Quote already finalised’ or ‘Quote not valid for Schm/Cat’.
  • Calculation Trace report - This report will display the quotation formulae that have been processed and is often a very useful debugging tool.

The system will then return to the main menu.

Exit Program

Click ‘Exit’. This leaves the program and closes the current browser tab.
