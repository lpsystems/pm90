Introduction

This program allows the user to store the Pension Master alternative to the data used in the external data source file. By entering in the required details through the selection criteria, the program will allow the user to translate the external data into Pension Master data.

The following options and screen appear when this program is selected:

  • Add
  • Query
  • Exit

Selection Screen

Add a Translation

Click ‘Add’.

Enter the selection criteria required.

Translation Code - Enter a translation code. This code will be used in conjunction with Rule 7 of the General Data Load program.

Translation Description - Enter a description.

Scheme Code - Enter a scheme code. Click the lookup icon for a list of valid scheme codes.

Import Data - Enter the data which is present in the external data file. The data will then be translated into the Pension Master data which has been entered in the next field.

Pension Master Data - Enter the Pension Master data which will be used to translate the Import Data.

Once this screen has been fully entered then click ‘OK’ to save the details.

Querying a Translation

Click ‘Query’.

The screen below will appear.

Enter the Scheme Code and/or Translation Code details. If any translation details match the entered criteria, then a pop-up window will appear listing them.

Select a translation and click ‘OK’ to display the details to the screen.

Updating a Translation

Click ‘Update’.

A translation code must have been selected before this option is used.

Enter the amended details and click ‘OK’ to save them.

Deleting a Translation

Click ‘Delete’.

A translation code must have been selected before this option is used.

If the translation code is used in the General Data Load program (Rule 7), then it cannot be deleted. The user will be warned of this.

Browsing a Translation

Click ‘Browse’ to view details in the screen where the data does not fit on the screen.

Once in browse mode, use the arrow keys or the scrollbar on the right of the data to navigate around the data.

Click ‘OK’ or ‘Cancel’ to leave the browse mode.

Reporting a Translation

Click the ‘Report’ icon.

The following window now appears:

Select Report Options Window

The options are as follows:

Print current record Produces a report of the translation details currently being viewed.
Print all selected record(s) Produces a report for all the translation codes selected.
Select record(s) to print Allows the user to select the records for which a report will be produced for.

Select the appropriate option and click ‘OK’. Click ‘Cancel’ to exit the printing or ‘Help’ to display this help text.

Exit Program

Click ‘Exit’. This leaves the program and closes the current browser tab.
