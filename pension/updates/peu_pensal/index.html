Index

Introduction

This program calculates pensionable salary for each selected member of a scheme based on gross salary and a deductible supplied by the user. The following options and screen appear when this program is selected:

  • Run
  • Exit

This program will overwrite whatever pensionable salary figures are in each of the selected members records for the given salary date with the new calculated value.

Run

Click ‘Run’.

Update Pensionable Salaries Screen

SELECTION CRITERIA - Select the group of members you require to update pensionable salaries for using the Scheme, Company, Category and Status fields (see Selection Criteria). When the selection criteria have been entered click on ‘OK’ to proceed.

You can click on ‘Cancel’ to return to the main menu and abort the update.

Update Pensionable Salaries Screen

SALARY DATE - Enter the Salary Date in the format dd/mm/yyyy or ddmmyy. If you do not enter a valid date or leave this field blank, an error message will be displayed and you will be requested to re-enter the date. This is the date of the salary entry on each of the selected members records you wish to change.

ORIGINAL SALARY - Enter any one of the salaries that have been set up for this scheme. To display a list of available salaries click on the lookup icon. The salary that you select here will be used to calculate the pensionable salary by having the deductible value subtracted from it.

DEDUCTIBLE - Enter the deductible you wish to apply to each selected members gross salary. If you leave this field blank it will default to 0.

NEW SALARY - Enter any one of the salaries that have been set up for this scheme. To display a list of available salaries click on the lookup icon. The salary that you select here should be the one that has been set up as pensionable salary. This salary will be replaced by the figure calculated by subtracting the deductible value from the Original Salary.

Click ‘OK’ and you will be asked to confirm that you wish to process the pensionable salary update for the category.

Updating Pensionable Salaries … Please Wait

When the pensionable salaries have been calculated for all members in the category the program will return to the main menu.

Click on ‘Cancel’ to return to the main menu without processing the pensionable salary updates.

Exit Program

Click ‘Exit’. This leaves the program and closes the current browser tab.
