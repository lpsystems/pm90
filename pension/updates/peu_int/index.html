Contribution Interest Run

Introduction

This program can either calculate interest on contributions, or remove contribution interest records for a specified group of members. This is used for certain types of defined benefit scheme, usually where there is a return of member contributions with interest to date on death or early withdrawal from the scheme.

The following options and screen appear when this program is selected:

  • Run
  • Undo
  • Exit

Contribution Interest Run Screen

NOTES

  1. It is intended that this program be run once a year at renewal time to update the interest on contributions for each member in the scheme. This should generally be done after you have loaded the last years contribution details onto each members contribution history record.
  2. The Undo process will actually remove the contribution interest records for the selected members. Please ensure that this is what you want to do, as this process is irreversible.

Run

Click ‘Run’.

You will be prompted to select a Provisional or Actual run.

Provisional This allows an interest run to be performed but does not update the Contribution & Interest Details.
Actual This allows an interest run to be performed and does update the Contribution & Interest Details.

SELECTION CRITERIA - Select the group of members you require to run the Contribution Interest Run for using the Scheme, Company, Category, Status, Special Code and Member fields. When the selection criteria have been entered click on ‘OK’ to proceed or click on ‘Cancel’ to return to the main menu.

A window is displayed asking for the run date for your annual interest posting. This will normally be the same as the scheme renewal date. Enter a date and click on ‘OK’ to proceed or ‘Cancel’ to return to the main menu.

The interest amounts will now be calculated and a progress meter is displayed informing you of calculation progress.

A report may be generated after the calculation of the annual interest amounts:

EXCEPTION REPORT - If any exceptions have been found then the number of exceptions will be displayed and you will be asked if you wish to view or print the report.

This report details any problems that have been found during the calculation of the annual interest amounts. Details are printed describing the problem that has been found. Possible exception messages are:

  • The last interest run for a member is on or after the date of the current interest run i.e. you are only allowed to do interest runs for dates after you last ran an interest run for the member.
  • The formula ‘incr’ has not returned a numeric value.
  • The formula ‘incr’ has returned a value greater than 100%.

If any exceptions are found then the annual interest amounts will not be saved.

RUN REPORT - If exceptions have not been found then a run report will be displayed to the screen.

This report details the annual interest amounts which will be saved to each member’s contribution & interest details screen in Members.

The options for printing are described in the document Printing From Pension Master.

When the interest has been calculated for all members in the selection criteria the program will return to the main menu.

Undo

Click ‘Undo’.

SELECTION CRITERIA - Select the group of members you require to remove Contribution Interest records for using the Scheme, Company, Category, Status, Special Code and Member fields. When the selection criteria have been entered click on ‘OK’ to proceed or click on ‘Cancel’ to return to the main menu.

A window is displayed asking for the run date you wish to undo interest records for. Enter a date and click on ‘OK’ to proceed or ‘Cancel’ to return to the main menu.

The interest records will now be selected and a progress meter is displayed informing you of the undoing progress.

A report may be generated after the selected interest records have been found:

UNDO REPORT - If any records have been found, then a report will be displayed to the screen.

This report details all records that will be removed during this process. The records listed will be for either on or before the run date entered by the user.

The options for printing are described in the document Printing From Pension Master.

Exit Program

Click ‘Exit’. This leaves the program and closes the current browser tab.
