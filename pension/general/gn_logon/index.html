Introduction

You will need to logon to Pension Master with a valid Pension Master username and password.

Logging on to Pension Master

The following screen allows you to log onto Pension Master using a valid Pension Master username and password.

Logon

Enter the fields as follows:

DATABASE - Select the name of the Pension Master database that you wish to connect to or select the Default Pension Master Database.

USER NAME - Enter your username as assigned to you by your System Administrator.

PASSWORD - Enter your password.

Once sufficient log in details have been entered, you will be granted access to Pension Master.

If there are any diary items for you, the Diary File Maintenance screen will now be shown. Once you have finished with the diary file, click ‘Exit’.

Pension Master Main Menu

The Pension Master main menu is now displayed:

Main Menu

The options you see on this menu will depend on those set up for you by your System Administrator but would typically be those shown in the screenshot above.

To expand a branch of the menu structure click on the disclosure triangle to the left of the menu option.

Menu options are indicated by a dot to the left of the option description. Select a menu option by double-clicking on the menu option text.

Menu

You can access the quick menu by pressing ‘M’ on the main screen. This will open a search function where you can type the name of the screen you’re looking for and go directly to it, without navigating through the full menu.

Quick Menu

Favourites

This feature allows the user to access and setup menu options which they most commonly use.

Favourites

Release Notes

This section provides a list of all updates and enhancements made to Pension Master since the user’s last logon, allowing users to stay informed about the latest changes.

Help

This option directs you to the user manual, opening directly at the section relevant to the screen you are currently viewing

About

The About section displays key technical and licensing details for Pension Master. Users may be asked to refer to this page when contacting support, especially for issues related to version numbers, session details, or licensing. It also provides quick access to download log files.

Logoff

Select this option securely log out of Pension Master and return to the Logon screen.
