Introduction

Only users who belong to a User Group that has the correct permission can access this option.

If you do not have permission, you will not be permitted access to this option.

Overview

The Diary Entry feature in Pension Master allows you to record and manage notes or reminders related to specific schemes, companies, or members. This function helps in tracking important events, actions, or follow-ups within the system.

The following options and screen appear when this program is selected.

  • Add
  • Query
  • Update
  • Move
  • Report
  • Browse
  • Exit

Diary Entry Details Screen

This screen shows all of your diary entries for diary dates less than or equal to today’s date. Only items with a status flag of ‘New’ or ‘Work-In-Progress’ will be displayed.

To display other diary entries e.g. for jobs that you have flagged as completed or for diary entries with dates in the future, you should use the Query option detailed below.

Add a Diary Item

Click ‘Add’. The following screen is displayed:

Add A Diary Item Screen

Date - Enter the date you wish the diary item to be posted to. You should note that the diary system will only display entries for dates less than or equal to today’s date (unless the record has been selected through the Query option - see below).

Time - Enter the time you wish the diary item to be posted to. You should note that if this field is left blank, then it will default to the current time.

Source - Select one of the following options from the drop-down list:

  • Member File
  • Scheme File
  • Company File
  • Category File
  • General - i.e. a diary entry not specific to a member, scheme, company or category.
  • Internet

Reference - Enter a reference number for your diary entry. The format of this depends on how you enter the Source field as follows:

  • AAAAA - if Source is Member File
  • BBBB - if Source is Scheme File
  • BBBB/CCCC - if Source is Company File
  • BBBB/DDD - if Source is Category File
  • 0 - if Source is General

where AAAAA = member number, BBBB = scheme code, CCCC = company code and DDD = category code e.g. for category 001 in scheme 0052, enter a Reference field value of 0052/001.

Action - Enter the text of the diary entry.

Status - Select one of the following options from the drop-down list:

  • Completed
  • New
  • Work-In-Progress

After the status has been entered the following window will be displayed, allowing the user to enter a memorandum.

Enter a memorandum

When the diary entry has been entered click ‘OK’ to save the details.

The cursor will now move into the lower half of the screen where you are invited to enter the usernames of each Pension Master user you wish the diary entry to be posted to. As you type in each user name, Pension Master will display that users full name to the screen. If you cannot remember a users username, click the lookup icon for a list of available users.

When the user names have been entered click ‘OK’ to save the details to the diary file. You can click ‘Cancel’ at any stage to exit the add option without saving the details.

Query/Search for a Diary Item

Click ‘Query’.

Query/Search For A Diary Item Screen

Type in the criteria to select the diary entry you wish to view and click ‘OK’. You must enter at least a user name and date. The diary details for that user and for all diary entry dates less than or equal to the date entry will then be displayed. You can optionally use the ‘STATUS’ field to tell Pension Master to only retrieve those records of a certain status.

You can click ‘Cancel’ at this stage to exit the Query option without proceeding.

If a match to the selection criteria is found the details will be displayed to the screen.

If no records are found to match the selection criteria, a message is displayed to the screen and the user is requested to re-enter the selection criteria.

Update a Diary Item

Click ‘Update’.

Move the cursor to the row to be updated and type over the entry with the new details. The fields are the same as those described in the section Add A Diary Item above.

You can click ‘OK’ to save the details or click ‘Cancel’ to exit without saving the details.

Move a Diary Item

Click ‘Move’ menu option.

You will be moved to the screen showing the diary entries for the current user. Highlight the diary entry you wish to move and click ‘OK’.

A list of users will be displayed. Highlight the user you wish to move the diary entry to and click ‘OK’.

The diary entry is moved to the specified user and the diary entries for the current user will be re-displayed to the screen.

Printing Diary Details

The options for printing are described in the document Printing From Pension Master.

Browsing Diary Details

Click ‘Browse’ to view details in any screen where the data does not fit on the screen. Once in browse mode, use the arrow keys or the scrollbar on the right of the data to navigate through the data.

Click ‘Memo’ to display a pop-up window displaying the Memo Entry. Click the ‘Printer’ icon to print the diary entry details and memo.

Click ‘OK’ or ‘Cancel’ to leave browse mode.

Exit Program

Click ‘Exit’. This leaves the program and closes the current browser tab.
