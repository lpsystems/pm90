Introduction

Only users who belong to a User Group that has the correct permission can access this option.

If you do not have permission, you will not be permitted access to this option.

This program allows maintenance of the workflow system in Pension Master. This allows you to set up groups of Workflow Actions which are known as Workflow Types. These are similar to the Quotation Type feature of Categories in that they are a way of grouping together similar actions.

If you have indicated in System Parameters that you wish to disable Workflow then this option will not be available.

You can allocate as many Workflow Actions to each of your Workflow Type as you wish, each action having a set number of days allocated against it as defined in the Action Codes program. When the user changes a member’s status code in Members, for example, when a member has left the scheme, a pop-up window is displayed which asks you for a Workflow Type code (or leave blank if you do not wish to use the Workflow System). Enter the appropriate Workflow Type code here, or click on the lookup icon to see a list of Workflow Type codes.

Pension Master then writes a set of entries to your diary, each dated as TODAY + number of days for the corresponding action code e.g. if today is 1/10/97 and I allocate the Workflow Type shown in the screenshot below (REFUND), the following entries will be written to the diary file for the following dates: 4/10/97, 5/10/97 and 15/10/97.

These entries will then appear on the said dates when the user logs on to Pension Master and serve as a reminder to them that deadlines for certain actions have passed. The System Administrator can use the Workflow Management Report to analyse who has what workflow entries in their diaries outstanding and hence identify possible bottlenecks in the administration department.

The following options and screen appear when this program is selected:

  • Add
  • Query
  • Update
  • Delete
  • Report
  • Exit

Workflow Management Screen

Add A Workflow Type

Click ‘Add’.

WORKFLOW CODE - Enter a code to be used for this Workflow Type. This can be up to 10 characters long and must not already exist on the Workflow Type file. If it does, an error message will be displayed and you will be invited to re-enter the Workflow Type code.

DESCRIPTION - Enter a description for your Workflow Type code. We would recommend that you use something meaningful here as this is what is displayed to users in any look windows for Workflow Type codes elsewhere in Pension Master. This should be a description of your intended group of actions e.g. “Refund Of Contributions”.

ACTION/TIME (DAYS) - Enter the individual Action Codes as appropriate for your Workflow Type. If you are unsure of the codes to use, click on the lookup icon to see a list of action codes. Once the action code has been entered, the action description and deadline in days is automatically displayed to the screen. You can amend the action deadline if you so wish by overwriting the value on the screen.

When the Workflow Type details have been entered click on ‘OK’ to save the details to file. You can click on ‘Cancel’ at any stage to exit the add option without saving the entered Workflow Type details.

Query/Search For A Workflow Type

Click ‘Query’.

You may select records using any of the fields on the screen.

Once you have entered your selection criteria, click on ‘OK’ to proceed and retrieve the records from the database or click on ‘Cancel’ to abort the selection and return to the main menu.

If any match to the selection criteria is found then the first record will be displayed to the screen and a message displayed to the bottom of the screen.

Viewing Workflow Types

To view the other records selected press N to view the next record, and P to view the previous record. If N is chosen and there are no records after the current record an error message is displayed. If P is chosen and there are no previous records an error message is displayed.

Updating Workflow Types

Before Updating, a record must be selected through the Query option, and an error message is displayed if a record for update has not been selected.

Click ‘Update’.

The Workflow Type code cannot be changed as this is a key to the file. When the changes have been made to the record click on ‘OK’ to save the changes to file, or click on ‘Cancel’ to exit the update option without saving changes.

Deleting Workflow Types

Before Deleting, a record must be selected through the Query option, and an error message is displayed if a record for deletion has not been selected.

Click ‘Delete’.

When the delete option is selected a window appears with the options ‘Yes’ and ‘No’.

If you do not wish to delete the record, click ‘No’. You will return to the program menu.

If you wish to delete the record, click ‘Yes’. A new window is displayed asking if you are sure.

If you do not wish to delete the record, click ‘No’. The program returns to the main menu.

If you wish to delete the record click ‘Yes’. The record is deleted and the program returns to the main menu.

Printing Workflow Type Details

Click the ‘Report’ icon.

If records have been selected in the program before the report option was selected the following window is displayed.

Print Options Screen 1

If no records have been selected in the program before the report option was selected the following window is displayed.

Print Options Screen 2

Record Prints the current screen Record.
Query Prints the all the selected records.
Select Allows selection of records for the report using Workflow Code and Workflow Description fields.
Exit Closes the window and leaves the report option.

If the Select option was chosen and no records are found to match the selection criteria entered, an error message is displayed and new selection criteria should be entered.

If records are found to match the selection criteria the report continues and options are then displayed for printing.

The options for printing are described in the document Printing From Pension Master.

Exit Program

Click ‘Exit’. This leaves the program and closes the current browser tab.
