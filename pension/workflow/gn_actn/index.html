Introduction

Only users who belong to a User Group that has the correct permission can access this option.

If you do not have permission, you will not be permitted access to this option.

This program allows maintenance of the workflow action codes.

These are codes that can be used to define procedures in Workflow Management.

The following options and screen appear when this program is selected:

  • Add
  • Query
  • Update
  • Delete
  • Report
  • Exit

Action Codes Screen

Add An Action Code

Click ‘Add’.

ACTION CODE - Enter a code to be used for this action code. This can be up to 10 characters long and must not already exist on the action codes file. If it does, an error message will be displayed and you will be invited to re-enter the action code.

DESCRIPTION - Enter a description for your action code. We would recommend that you use something meaningful here as this is what is displayed to users in any look windows for action codes elsewhere in Pension Master. This should be a description of your intended action e.g. “Pay Tax On Refund Of Contributions”.

TIME (DAYS) - Enter the time in days before which this action should be completed from inception.

When the Action Code details have been entered click on ‘OK’ to save the details to file. You can click on ‘Cancel’ at any stage to exit the add option without saving the entered Action Code details.

Query/Search For An Action Code

Click ‘Query’.

You may select records using any of the fields on the screen.

Once you have entered your selection criteria, click on ‘OK’ to proceed and retrieve the records from the database or click on ‘Cancel’ to abort the selection and return to the main menu.

If any match to the selection criteria is found then the first record will be displayed to the screen and a message displayed to the bottom of the screen.

Record 1 of 2

Viewing Action Codes

To view the other records selected press N to view the next record, and P to view the previous record. If N is chosen and there are no records after the current record an error message is displayed. If P is chosen and there are no previous records an error message is displayed.

Updating Action Code

Before Updating, a record must be selected through the Query option, and an error message is displayed if a record for update has not been selected.

Click ‘Update’.

The Action Code code cannot be changed as this is a key to the file. When the changes have been made to the record click on ‘OK’ to save the changes to file, or click on ‘Cancel’ to exit the update option without saving changes.

Deleting Action Code Details

Before Deleting, a record must be selected through the Query option, and an error message is displayed if a record for deletion has not been selected.

You can not delete an action code that is currently being used in some workflow type definition (see Workflow Management for further details).

Click ‘Delete’.

When the delete option is selected a window is displayed with the options yes and no.

If you do not wish to delete the record, click ‘No’. You will return to the program menu.

If you wish to delete the record click ‘Yes’.

Another window is displayed asking if you are sure.

If you do not wish to delete the record click ‘No’.

The program returns to the main menu.

If you wish to delete the record, click ‘Yes’.

The record is deleted and the program returns to the main menu.

Printing Action Code Details

Click the ‘Report’ icon.

A record must be selected before the report option is selected, otherwise the following error message will be displayed to the screen.

No Details Selected For Report. Please Use Query Option First.

When the report option is selected after a record has been queried, the user is given three printing options as described in the document Printing From Pension Master.

If ‘Select records to print’ is selected, the following window is displayed.

If records are found to match the selection criteria the report continues and the report viewer is displayed to the screen.

Exit Program

Click ‘Exit’. This leaves the program and closes the current browser tab.
