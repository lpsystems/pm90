Introduction

Only users who belong to a User Group that has the correct permission can access this option.

If you do not have permission, you will not be permitted access to this option.

This program allows the user to keep track of Transaction Logs made by members of Pension Master via Pension Master Online.

The following options and screen appear when this program is selected:

  • Query
  • Report
  • Browse
  • Exit

Transaction Log Listing Screen

Selecting Transaction Log Details

Click ‘Query’.

You may select records by using any combination (or even partial combination) of the following fields in order to select the required transaction log record(s): Log ID, Member, Date or Time.

Once you have entered your selection criteria, click on ‘OK’ to proceed and retrieve the records from the database or click on ‘Cancel’ to abort the selection and return to the main menu.

If any match to the selection criteria is found then the first record will be displayed to the screen and a message displayed to the bottom of the screen.

Record 00001 of 0000x

Printing Transaction Log Details

Click the ‘Report’ icon.

You must have selected some records to print first by using the Query menu option.

The following menu is now displayed:

REPORT TYPE: Summary Detailed Record Exit

The menu options are as follows:

Summary Produce a report for all selected items in summary format.
Detailed Produce a report for all selected items in detailed format.
Record Produce a report for the current displayed item only.
Exit Return to the main menu.

Once the type of report has been chosen and produced, options are then displayed for printing.

The options for printing are described in the document Printing From Pension Master.

Browsing Transaction Log Details

Click ‘Browse’.

You must have selected some records to browse first by using the Query menu option.

Once in browse mode, use the arrow keys or the scrollbar on the right of the data to navigate around the data.

Click ‘OK’ or ‘Cancel’ to leave browse mode.

Exit Program

Click ‘Exit’. This leaves the program and closes the current browser tab.
