Introduction

Only users who belong to a User Group that has the correct permission can access this option.

If you do not have permission, you will not be permitted access to this option.

This program allows maintenance of the User Group file. This allows the System Administrator to control who is allowed access to Pension Master and what options they can access e.g. they can set a user group up so that they do not have access to the more sensitive areas of the system like Formulae.

The following options and screen appear when this program is selected:

  • Add
  • Query
  • Update
  • Delete
  • Report
  • Menu
  • Exit

User Group File

Add a User Group

Click ‘Add’.

CODE - Enter the Code. This can be a maximum of 10 alphanumeric or numeric characters.

DESCRIPTION - Enter the description of the usergroup.

PERMISSION - This allows you to set up the menu options that will be available for this user.

The programs are listed on the left hand side and if want the user group to have access you tick the permission box. On the bottom of the screen you are given the option of all or none. Programs not listed here for example Basic Member Details are is an area that all users have general access unless you switch it off within the actual user file.

A User Group can also be copied.

Query/Search For A User Group

Click ‘Query’.

If any match to the selection criteria is found then the first record will be displayed to the screen and a message displayed to the bottom of the screen.

Record 1 of 2

Browsing User Group Details

Click on the ‘Browse’ button or the Browse icon on the toolbar to view details in any screen where the data does not fit on the screen.

Once in browse mode, use the arrow keys or the scrollbar on the right of the data to navigate around the data.

Click on ‘OK’ or ‘Cancel’ to leave browse mode.

Viewing User Group Records

To view the other records selected press N to view the next record, and P to view the previous record. If N is chosen and there are no records after the current record an error message is displayed. If P is chosen and there are no previous records an error message is displayed.

Update A User Group Record

Before Updating, a record must be selected through the query option, and an error message is displayed if a record for update has not been selected.

Click ‘Update’.

The UserGroup Code field cannot be changed as this is the key to the user group file. When the changes have been made to the record click on ‘OK’ to save the changes to file, or click on ‘Cancel’ to exit the update option without saving changes.

Delete A User Group Record

Before Deleting, a record must be selected through the query option, and an error message is displayed if a record for deletion has not been selected.

Click ‘Delete’.

When the delete option is selected a window appears with the options ‘Yes’ and ‘No’.

If you do not wish to delete the record, click ‘ No ‘. You will return to the program menu.

If you wish to delete the record, click ‘Yes’. A new window is displayed asking if you are sure.

If you do not wish to delete the record, click ‘No’. The program returns to the main menu.

If you wish to delete the record, ‘Yes’. The record is deleted and the program returns to the main menu.

You are unable to delete a user group if there are users still assigned to it under the user file.

Printing User Details

Click the ‘Report’ icon.

If records have been selected in the program before the report option was selected the following window is displayed.

Print Options Screen 1

If no records have been selected in the program before the report option was selected the following window is displayed.

Print Options Screen 2

Record Prints the current screen Record.
Query Prints the all the selected records.
Select Allows selection of records for the report using User ID or User Name.
Exit Closes the window and leaves the report option.

If the Select option was chosen and no records are found to match the selection criteria entered, an error message is displayed and new selection criteria should be entered.

If records are found to match the selection criteria the report continues and options are then displayed for printing.

The options for printing are described in the document Printing From Pension Master.

Exit Program

Click ‘Exit’. This leaves the program and closes the current browser tab.
