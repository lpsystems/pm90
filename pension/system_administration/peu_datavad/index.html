Introduction

This program allows the user to analyse member address data, investment fund data and data held in any of the status history screens. You can select one or all the reports to run at the same time. The reports are produce into pdf’s and list the following for each report :

Member Address Validation Member XXXX has more than one current address. Member XXXX only has one address and it’s not current. Member XXXX has more than one address and none of them are current.

Investment Fund Price Validation Fund XXXX has a zero price with the Buy Basis set to Bid for DD/MM/YYYY. Fund XXXX has a zero price with the Sell Basis set to Bid for DD/MM/YYYY. Fund XXXX has a zero price with the Buy Basis set to Mid for DD/MM/YYYY. Fund XXXX has a zero price with the Sell Basis set to Mid for DD/MM/YYYY. Fund XXXX has a zero price with the Buy Basis set to Offer for DD/MM/YYYY. Fund XXXX has a zero price with the Sell Basis set to Offer for DD/MM/YYYY.

Member Status/Company/Category/Marital Status History Validation Member XXXX, Test Member has no Status/Company/Category History. Member XXXX, Test Member has more then 1 Status/Company/Category History record at the same effective date and at least 1 of the records has a 0 background id. Member XXXX, Test Member current status/Company/Category code doesn’t match with the top line of their Status History/Company/Category code Member XXXX, Test Member has a line in their Status/Company/Category History where the New code equals the Old code. Member XXXX, Test Member has a line in their Status/Company/Category History where the New code doesn’t match the Old code of the line above it. Member XXXX, Test Member last line of their Status/Company/Category History doesn’t have an Old code of ‘NEW’.This program deletes a member’s special code in bulk relating to the standard selection criteria screen of scheme, company, category, status and group. For all members in the selection if they have the special code assigned to them they are produced out to a report before the user is asked if they wish to continue and delete the special codes from these members.
