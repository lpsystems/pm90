Introduction

Only users who belong to a User Group that have the correct permission can access this option.

If you do not have permission, you will not be permitted access to this option.

This program allows maintenance of the menu options. Before an item can be displayed in the menus it must be set up on the menu options file. The following options and screen are displayed when this program is selected:

  • Add
  • Query
  • Update
  • Delete
  • Exit

Menus Screen

Add a Record

Click ‘Add’.

The field Module Id is a unique reference code to the menu options file and is automatically generated when the add option is selected.

MODULE TYPE - The valid entries for this field are:

  • PENSION - Relating to Pension Administration
  • PAYROLL - Relating to Payroll Administration
  • GENERAL - General (system administration or none of the above)

OPTION TYPE (PROGRAM/MENU) -The valid entries for this field are:

  • PROGRAM - Denotes that menu option is a program
  • MENU - Denotes that menu option is a menu or sub-menu

DESCRIPTION - This is the program/menu description that will appear on the menus. The description entered must be unique to the module. ie. Two records with module type ‘PE’ and a description of ‘Members’ cannot exist.

OPTION RUN COMMAND - If the menu option is a menu/sub-menu the field ‘Option Run Command’ does not have to be filled. If the menu option is a program then the command that would run the program should be entered here. e.g. pem_mem.4ge. If you do not fill this in correctly, an error message will be displayed when any of your users try to use this menu option and the appropriate module will not run.

When the Menu Option details have been entered click on ‘OK’ to save the details to file. You can click on ‘Cancel’ at any stage to exit the add option without saving the entered details.

Query a Record

Click ‘Query’.

If any match to the selection criteria is found then the first record will be displayed to the screen and a message displayed to the bottom of the screen.

Record 1 of 2

Viewing Records

To view the other records selected press N to get the next record, and P to view the previous record. If N is chosen and there are no records after the current record and error message is displayed. If P is chosen and there are no previous records an error message is displayed.

Update a Record

Before Updating, a record must be selected through the query option, and an error message is displayed if a record for update has not been selected.

Click ‘Update’.

The menu option Id cannot be changed as this is a key to the menu option file. When the changes have been made to the record click on ‘OK’ to save the changes to file, or click on ‘Cancel’ to exit the update option without saving changes.

Delete a Record

Before Deleting, a record must be selected through the query option, and an error message is displayed if a record for deletion has not been selected.

Click ‘Delete’.

When the delete option is selected a window appears with the options ‘Yes’ and ‘No’.

If you do not wish to delete the record, click ‘No’. You will return to the program menu.

If you wish to delete the record, click ‘Yes’. The following menu is displayed

If you do not wish to delete the record click ‘No’. The program returns to the main menu. If you wish to delete the record, click ‘Yes’.

If the menu option being displayed is currently displayed on any menu in the system it will not be displayed on the menu after deletion.

The record is deleted and the program returns to the main menu.

Exit Program

Click ‘Exit’. This leaves the program and closes the current browser tab.
