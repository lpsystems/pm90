Introduction

Only users who belong to a User Group that has the correct permission can access this option.

If you do not have permission, you will not be permitted access to this option.

This program allows maintenance of the Audit Trail. This is where all changes to data on Pension Master get logged. The amount of detail logged is dependent on how you set the audit trail flag up in System Parameters.

It is good practice to print off and clear down the audit trail on either a weekly or monthly basis so as to avoid the audit trail becoming too large.

The following options and screen appear when this program is selected:

  • Query
  • Clear
  • Exit

Audit Trail Listing Screen

Selecting Audit Trail Details

Click ‘Query’.

You may select records by using any of the fields on the screen. You may also use any combination (or even partial combination) of these fields in order to select the required company record(s).

Once you have entered your selection criteria, click on ‘OK’ to proceed and retrieve the records from the database or click on ‘Cancel’ to abort the selection and return to the main menu.

If any match to the selection criteria is found then the first record will be displayed to the screen and a message displayed to the bottom of the screen.

Record 00001 of 0000x

Clearing Audit Trail Details

Click ‘Clear’.

This option allows you to delete Audit Trail Listing data by selecting the oldest information first.

The oldest date will be displayed as the start date, you can then enter the end date of the range of information to be deleted.

Clear Date Entry

A report will be produced containing the selected information.

You will then be asked to confirm that you wish to delete the data.

This action is not reversible. The audit trail records selected will be permanently deleted from the database.

Clear Confirmation

Upon confirmation a message will be displayed indicating that the data has been deleted.

Printing Audit Details

Click the ‘Report’ icon.

You must have selected some records to print first by using the Query menu option.

Once the type of report has been chosen and produced, options are then displayed for printing.

The options for printing are decribed in the document Printing From Pension Master.

Exit Program

Click ‘Exit’. This leaves the program and closes the current browser tab.
