Introduction

Only users who belong to a User Group that has the correct permission can access this option.

If you do not have permission, you will not be permitted access to this option.

This program deletes all details relating to a specified scheme from the Pension Master database and optionally saves the deleted details to disk. The following options and screen appear when this program is selected:

  • Proceed
  • Exit

Delete a Scheme Screen

Delete a Scheme

Press P, or select the ‘Proceed’ menu option.

SCHEME - The scheme code must be entered and must exist on the scheme file. When a valid code has been entered the scheme name will be displayed.

CONFIRM SCHEME - Re-enter the scheme code. The scheme code must be entered and must be the same code as previously entered. This is to guard against any typographical errors. The scheme name will be displayed when a valid scheme code has been entered.

INCLUDE HEADING - Headers will be included in the output.

DELETE DATA - All data relating to the selected scheme will be deleted from Pension Master.

When all the member details have been deleted a message will be displayed and the program will return to the main menu.

Exit Program

Click ‘Exit’. This leaves the program and closes the current browser tab.
