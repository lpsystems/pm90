Introduction

Only users who belong to a User Group that has the correct permission can access this option.

If you do not have permission, you will not be permitted access to this option.

This program allows you to print out the details needed for L&P Systems Limited quarterly fee invoices and should only be run when requested to by a member of L&P Systems Limited staff.

The following options and screen appear when this program is selected:

  • Run
  • Exit

Quarterly Fees Print Screen

Running The Report

Click ‘Run’.

INVOICE DATE - Enter an invoice date. The number of members held on Pension Master, as at the invoice date entered, will be calculated. Any members added to Pension Master after this date will not be included in the quarterly fees report.

Once the report has been produced, options are displayed for printing.

The options for printing are described in the document Printing From Pension Master.

Exit Program

Click ‘Exit’. This leaves the program and closes the current browser tab.
