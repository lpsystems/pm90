Introduction

Only users who belong to a User Group that has the correct permission can access this option.

If you do not have permission, you will not be permitted access to this option.

This program allows maintenance of the menus within Pension Master. You use this program to setup what options appear on which menus so as you can customise how Pension Master appears to your users. Pension Master is supplied as standard with a default set of menus.

The following options and screen are displayed when this program is selected:

  • Update
  • Exit

Menus Screen

Update a Record

Click ‘Update’.

Click the lookup icon for a list of menus and menu options.

Once the appropriate menu has been selected, click on ‘OK’ to select this or click on ‘Cancel’ to exit the update option without saving any changes.

The cursor then moves down into the body of the menu. Click on ‘Add Row’ to insert a blank line or click on ‘Delete Row’ to delete a line. You can click on the lookup icon to see a list of menu options.

When you have finished making alterations to the menu, click on ‘OK’ to save the menu to file or click on ‘Cancel’ to exit the update option without making any changes.

Exit Program

Click ‘Exit’. This leaves the program and closes the current browser tab.
