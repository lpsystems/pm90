Introduction

This program produces a report of membership movements between different Status Codes for a selected group of members between two dates entered by the user. This is done by interrogating the Status History file for each selected member.

This report is produced by interrogating the status history records for each selected member between two dates. If the status history records are incorrect (e.g. if a user has deleted part of a members status history in Members), the report may show inconsistencies. We would therefore recommend that this report be run in conjunction with the members joined, members left and impending retirements reports so as to provide a cross-check.

The following options and screen appear when this program is selected:

  • Proceed
  • Exit

Member Selection Screen

Run A Report

Click ‘Proceed’.

Enter the selection criteria required (see Selection Criteria). When the selection criteria has been entered click on’OK’.

You can click on ‘Cancel’ to return to the proceed menu and abort report selection.

When this field has been entered a second screen is drawn as follows:

Date Selection Screen

REPORT DATE - The report selection dates default to producing the report for the year immediately prior to today’s date, but can be overwritten by any dates in the format dd/mm/yyyy or ddmmyyyy.

You can click on ‘Cancel’ to exit input of report dates and return to the report menu.

When two dates have been entered the following menu is displayed:

REPORT TYPE: Summary Detailed Exit

The menu options are as follows:

Summary Produce a report showing only the numbers of members moving between different status codes for the selected group of members. Type S, or select the Summary menu option.
Detailed Produce a report showing the member details for each member moving between different status codes with the date that the status or the change date was updated for the selected group of members, together with totals for the number of movements. Type D, or select the Detailed menu option.
Exit Return to the main menu without processing the report. Type E, or select the Exit menu option.

A message is now displayed at the top of the screen:

Processing report … Please Wait

When the report has been processed and no records are found to match the selection criteria an error message is displayed:

There are no records to match the selection criteria

If records are found to match the selection criteria the report continues and options are then displayed for printing.

The options for printing are described in the document Printing From Pension Master.

Exit Program

Click ‘Exit’. This leaves the program and closes the current browser tab.
