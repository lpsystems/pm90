Introduction

Pension.txt Creator is a separate application that extracts pre determined field values directly from your pension master database. It was designed to extract the mail merge data for defined contribution schemes benefit statements. The tables it extracts data from are as follows :

  • Basic Member Address
  • Member Addresses
  • Scheme Addresses as indicated in the DCgeneric Set up.
  • Additional Scheme Addresses for the following:
  • Investment Managers that have the address code INVEST1 to INVEST5
  • Employers that have the address codes DCER1 to DCER3
  • Quotation Screens
  • Dcgeneric Conifigure fields for death in serviced multiples, spouses death in service benefit % and Disability Benefit

In relation to the quotation screens, it extracts all of the indexes in the quotation type that you have indicated. Make sure that you have run the quotes you want to extract results from before you use the pension.txt creator

Running

On your pc proceed as follows

Click Start All Programs Penmast.txt Creator

The following screen is displayed:

Pension Creator

SQL SERVER NAME - The SQL Server Name will be auto populated. It is the server where the pension creator programs reside. The name you insert here is saved to a configure file and will be the default for when you next use the creator.

DATABASE - You must select your database from the drop-down window. Generally, the database will be Pension and as such it appears first on the list

USERNAME & PASSWORD - Input your pension master user name and password and click the ok. The database you have selected will be checked for the user details you put in and access will only be granted if your user details are on the database.

The following screen is displayed:

selection Section

SELECT SCHEME - Use the drop-down window to select your scheme. The schemes are listed alphabetically by description. This field is mandatory

SELECT CATEGORY - Use the drop-down window to select your category. The categories are listed alphabetically by description. This field is mandatory. You only have the option of running the creator for one category or all categories.

SELECT QUOTATION - Use the drop-down window to select your quote. The quotes specific to the category selected are listed alphabetically by description. This field is mandatory.

TRANSACTION TYPE - Enter the transaction type for the contributions you are posting to units. If you are unsure of what code to use, click the lookup icon to see a list of transaction codes

If your quote has the ‘DCGENST’ flag field ticked, this field will be automatically ticked. If you do not want the DCGENST quote results, just untick this field. Include SORPS results (available for Irish schemes only and works the same as Include DCGENST results flag.

SELECTION CRITERIA SECTION

The selection criteria section allows you to select by company, status type and special code.

Use the drop-down window to make your selection. You only have the option of selecting one or all. If you leave these fields blank, it is assumed you have opted for All.

GROUP BY SECTION

There are 4 levels to group by and the field options are as follows -

  • Category
  • Company
  • Surname
  • Forename
  • Status Type
  • Staff Number
  • NI/PPS Number

If you leave the group by fields blank, the members will appear in surname, forename order in the file.

RUNNING THE FILE

When you have entered all the criteria on the screen, click the create button on the bottom right of the screen. You will be advised of the number of members processed and where the pension.txt file will reside. It is usually in c:\penmast on your PC.

The following message is displayed:

Message

USING THE PENSION.TXT FILE WITH YOUR BENEFIT STATEMENT TEMPLATE

Your benefit statement template will be looking for penmast.doc so you need to update the template to link to the correct data source file Using windows 2007 proceed as follows:

  • Click the mailings tab
  • Click the Select Recipients icon

The screen looks like this:

Mailing Tab

  • From the drop-down list select ‘use existing list’ and the window below will appear. Select My Computer in the left menu bar and find you pension.txt file (it will reside in the folder indicated in the ‘local path’ field in system parameters, usually c:\penmast) and click on it.

The screen looks like this:

Mailing Tab

  • You will now be back in your template. Save it so the new data source file is saved for all future use of the template.
  • Continue your mail merge in the usual manner

Exit Program

Click ‘Exit’. This leaves the program and closes application.
