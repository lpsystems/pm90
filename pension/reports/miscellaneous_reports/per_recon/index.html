Introduction

This program allows the user to produce a reconciliation of Scheme membership, salary and contribution data. The report will be of great use following the take on of new scheme data and will aid the checking of the loading of members and their salaries and contributions. The following options and screen appear when this program is selected:

  • Summary
  • Detailed
  • Exit

Selection Screen

Run A Report

If the Member option is selected, the generated report will display totals for each individual member for each of their salary and contribution elements along with a count of the number of lines of data present for each respective record. After this information has been extracted for the Schemes membership, a summary is shown for that Scheme detailing the total number of members and total salaries, contributions and lines of data.

The Scheme option will display simply display the summary totals for each extracted Scheme.

Click ‘Summary’ for a Scheme report or ‘Detailed’ for a Membership breakdown.

**Scheme - **Please enter the scheme that you wish to report on.

After selecting the scheme that you wish to report on, please tick up to five boxes to indicate which salary types you would like to report on.

When the report has been processed and no records are found to match the selection criteria an error message is displayed:

There are no records to match the selection criteria

If records are found to match the selection criteria the report continues and options are then displayed for printing.

The options for printing are described in the document Printing From Pension Master.

Exit Program

Click ‘Exit’. This leaves the program and closes the current browser tab.
