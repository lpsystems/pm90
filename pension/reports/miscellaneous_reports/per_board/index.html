Introduction

This program allows the user to produce the standard Pensions Board Fee Invoice for a selected scheme. This can be used to bill your scheme for the amount due in respect of the Pensions Board Fee for the current year.

This report is only applicable to Irish based schemes.

The following options and screen appear when this program is selected.

  • Proceed
  • Exit

Selection Screen

Run A Report

Click ‘Proceed’.

SCHEME CODE - Enter the scheme code for the scheme you wish to run the report for. If you are unsure of the scheme code, click on the lookup icon for a list of scheme codes.

DUE DATE - Enter the due date for the Pensions Board fees in the format dd/mm/yyyy or ddmmyyyy. This will be printed on the report. This filed will default to the next available renewal date, by reference to the Renewal Date field on the Basic Scheme Details screen in Schemes.

You can click on ‘OK’ to run the report, or click on ‘Cancel’ to abort the report.

When the selection is complete a message is displayed at the bottom of the screen:

Processing report … Please Wait

Once the report has been produced, options are displayed for printing.

The options for printing are described in the document Printing From Pension Master.

Exit Program

Click ‘Exit’. This leaves the program and closes the current browser tab.
