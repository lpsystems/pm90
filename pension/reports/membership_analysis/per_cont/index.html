Introduction

This program produces a report of contribution details as selected by the user for a given group of members, date due, date received, transaction type, investment fund and contribution type. You have the option of printing the report in Summary or Detail formats. The following options and screen appear when this program is selected:

  • Proceed
  • Exit

Contributions Report Screen

Run A Report

SELECTION CRITERIA - Select the group of members you require to allocate contributions for using the Scheme, Company, Category, Status, Special Code and Member Number fields (see Selection Criteria). When the selection criteria have been entered click ‘OK’ to proceed.

You can click ‘Cancel’ to return to the main menu and abort the contributions report.

The following screen is displayed:

Date Selection Screen

DATE DUE - Enter the date due for the contributions you are reporting on.

DATE RECEIVED - Enter the date you received the contributions you are reporting on.

TRANSACTION TYPE - Enter the transaction type for the contributions you are repoting on. If you are unsure of what code to use, click the lookup icon to display a list of transaction codes.

INVESTMENT FUND - Enter the investment fund for the contributions you are reporting on. If you are unsure of what code to use, click the lookup icon to display a list of transaction codes. If you want to report on the total investment funds then enter ‘TOT’.

REPORT TYPE - Enter the report type format (Summary or Detailed) you wish to report the contribution amounts in.

CONTRIBUTION TYPE - Enter the contribution type. Select ‘All’ if you wish to report on all contribution types for the scheme selected, or ‘Select’ if you wish to choose certain contribution types. The ‘Select’ option will pop up the following window:

The following screen is displayed:

Select Contribution Type Screen

Select the contribution types you wish to report on.

Once the report has been produced, options are displayed for printing.

The options for printing are described in the document Printing From Pension Master.

Exit Program

Click ‘Exit’. This leaves the program and closes the current browser tab.
