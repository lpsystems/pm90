Introduction

The following options and screen appear when this program is selected:

  • Proceed
  • Exit

Selection Screen

Run an Approaching Benefit Report

Click ‘Proceed’.

Enter the selection criteria required.

Scheme - This represents the code corresponding to the scheme the member belongs to. The scheme code must be entered and the code entered must exist on the scheme file. Click on the lookup icon to see a list of the scheme codes.

IFA - The IFA code must be entered and the combination of scheme and IFA code must exist on the IFA file. To see a list of the existing combinations of scheme and IFA codes on the IFA file click on the lookup icon.

Salesperson - The Salesperson code must be entered and the combination of scheme and Salesperson code must exist on the Salesperson file. To see a list of the existing combinations of scheme and Salesperson codes on the Salesperson file click on the lookup icon.

Status - This represents the current status of the member in the scheme, e.g. active member, deferred pension The status code entered must exist on the status file. To see a list of the existing status codes click on the lookup icon.

Special Code - Enter the special code as set-up in Codes.

Member

If no fields are entered then a message appears at the top of the screen asking you if you are sure that you want to select all of the members. The following message appears:

You have selected all members. This may take some time to retrieve from the database. Proceed anyway?

When the selection criteria has been entered click on ‘OK’ to proceed. You can click on ‘Cancel’ to return to the proceed menu and to abort the selection fields.

If no members can be found with the selection criteria you have entered, then a message window will be displayed informing you of this:

There are no records to match the report criteria.

Date Range Screen

Report Date - From - From date.

Report Date - To - To date.

Letter Selection Screen

Letter? - Select which members need to have a Benefit Quote Letter sent to them.

A list of the letters that will be produced will be displayed and you will be asked to confirm that you wish to continue.

If exceptions have been found then a report will be produced listing the exceptions that have been found.

Exit Program

Click ‘Exit’. This leaves the program and closes the current browser tab.
