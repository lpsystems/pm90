Introduction

This program produces a report of members status codes at a specified date in the past for a selected group of members. You have the option of doing this in either Summary or Detailed format.

The Summary Report will provide you with the number of members held under each new status code using the selection criteria and the effective date entered by the user.

The Detailed Report will provide you with details of each individual member held under each new status code, these including the member’s name and number, the effective date, old and new status codes, and the date when the status code was updated. This is also done by using the selection criteria and the effective date entered by the user.

The following options and screen appear when this program is selected:

  • Proceed
  • Exit

Selection Screen

Run a Historical Status Report

Click ‘Proceed’.

Enter the selection criteria required.

SCHEME CODE - This represents the code corresponding to the scheme the member belongs to. The scheme code must be entered and the code entered must exist on the scheme file. Click on the lookup icon to see a list of the scheme codes. When a valid scheme code has been entered the scheme name will be displayed.

COMPANY CODE - The company code must be entered and the combination of scheme and company code must exist on the company file. To see a list of the existing combinations of scheme and company codes on the company file click on the lookup icon. When a valid company code has been entered the company description will be displayed.

CATEGORY CODE - The category code must be entered and the combination of scheme and category code must exist on the category file. To see a list of the existing combinations of scheme and category codes on the category file click on the lookup icon. When a valid combination of scheme and category code has been entered the category description will be displayed.

STATUS CODE - This represents the current status of the member in the scheme, e.g. active member, deferred pension The status code entered must exist on the status file. To see a list of the existing status codes click on the lookup icon. When a valid status code has been entered the status description will be entered.

SPECIAL CODE - Enter the special code as set-up in Codes.

At least one of the above fields must be entered before you can enter the effective date. If no fields are entered then a message appears at the top of the screen asking you if you are sure that you want to select all of the members. The following message appears:

You have selected all members. This may take some time to retrieve from the database. Proceed anyway?

EFFECTIVE DATE - This represents the date at which the status will change. If the date entered is after today’s date i.e. in the future, then a message will be displayed prompting the user to check if this date is correct. This message is as follows:

The Effective Date entered is in the future. Are you sure that you want to use it as the Effective Date?

If no date is entered then the date will default to today’s date. A message appears at the top of the screen informing you of this:

An Effective Date has not been entered. The Effective Date will default to Today’s date. Proceed anyway?

When the selection criteria has been entered click on ‘OK’ to proceed. You can click on ‘Cancel’ to return to the proceed menu and to abort the selection fields.

If no members can be found with the selection criteria you have entered, then a message window will be displayed informing you of this:

There are no records to match the report criteria.

If members have been found then the following menu is displayed:

REPORT: Summary Detailed Exit

Here you have a choice of two reports depending on how much information you require.

To produce a Summarised Historical Status Report, click ‘Summary’.

To produce a Detailed Historical Status Report, click ‘Detailed’.

To exit the menu, click ‘Exit’.

The options for printing are described in the document Printing From Pension Master.

Exit Program

Click ‘Exit’. This leaves the program and closes the current browser tab.
