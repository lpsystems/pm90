Introduction

This program produces a report out to Microsoft Excel of contribution details, as selected by the user, for a given group of members, for either all contributions or selected contributions, between a date range entered. You have the option of printing a top header per member, and/or printing a footer per member. The following options and screen appear when this program is selected:

  • Proceed
  • Exit

Selection Criteria Screen

Run A Report

SELECTION CRITERIA - Select the group of members you require to produce this report for using the Scheme, Company, Category, Status, Special Code and Member Number fields (see Selection Criteria). When the selection criteria have been entered click ‘OK’ to proceed.

You can click ‘Cancel’ to return to the main menu and abort the contribution reconciliation report.

The following screen is displayed:

Date Selection Screen

CONTRIBUTIONS - Click ‘All’ if you wish to report on all the contributions for the scheme, or click ‘Select’ to select contributions to report on. Clicking ‘Select’ will pop-up a window as follows:

Select Contributions Screen

START DATE - Enter the start date for the contributions you are reporting on.

END DATE - Enter the end date for the contributions you are reporting on.

PRINT TOP HEADER PER MEMBER- Tick this field if you wish to have the standard top header “Contribution Reconciliation Report For scheme description” before every member’s details.

PRINT FOOTER PER MEMBER- Tick this field if you wish to print a footer after every member’s details. The footer can be entered in the following ‘Footer’ field.

FOOTER- This field is defaulted to “Note: No deductions or charges were applied to the amounts shown before investment, i.e. 100% of the amounts shown were invested.”. You can either leave it as this, or overwrite it with your own. This footer will only be printed if the field above is ticked.

When the details in this screen have been entered, click on ‘OK’ to proceed. You can click on ‘Cancel’ to return to the main menu and abort the contribution reconciliation report. The report will now be created and reported out to Excel.

Once the report has been produced, a message will appear to the screen stating that the report has completed successfully.

Exit Program

Click ‘Exit’. This leaves the program and closes the current browser tab.
