Introduction

This program allows the user to produce a listing on those members who have exceeded their contribution salary cap for the year leading up to a given date for the member selection criteria input. The report details member data, the total contributions, the annual salary and the % the actual total contributions received are of that annual salary.

For this report to run, certain fields in scheme file have to be updated. In the basic scheme details screen there are 3 fields: 1) Salary Frequency. The options for this field are monthly/annually to identify if the salary being recorded on the salary history screen is an annual or a monthly amount 2) Contribution Salary Cap. This is a % field to hold the cap %. 3) The contribution type set up screen in schemes has an ‘Include in Salary Cap’ indicator field for this purpose.

Contributions then by date received for the year leading up to and including the run date, for each contribution code where the ‘‘include in salary cap’ field is set to Y, are totalled together. The total contributions are checked to see if it is greater than the latest salary record in column 1 of salary history screen 1, at or effective at the run date, multiplied by the salary frequency in the basic scheme details screen multiplied by the salary cap % from the basic scheme details screen. Then if the total contributions are greater, the member is included in the report

Selection Criteria Screen

Run A Report

SELECTION CRITERIA - Select the group of members you require to produce this report for using the Scheme, Company, Category, Status, Special Code and Member Number fields (see Selection Criteria). When the selection criteria have been entered click ‘OK’ to proceed.

You can click ‘Cancel’ to return to the main menu and abort the contribution reconciliation report.

The following screen is displayed:

Date Selection Screen

This date is used as the effective date for the salary, and to establish the year to calculate the total contributions.

When the details in this screen have been entered, click on ‘OK’ to proceed. You can click on ‘Cancel’ to return to the main menu and abort the contribution reconciliation report. The report will now be created and reported out to Excel.

Once the report has been produced, a message will appear to the screen stating that the report has completed successfully.

Exit Program

Click ‘Exit’. This leaves the program and closes the current browser tab.
