Introduction

This program allows the user to produce a report that will give the total contributions posted for a scheme for a certain time period for all or one funds or transaction type compared to any of the dates on the investment history screen

  • Proceed
  • Exit

Selection Screen

Run A Report

Selection Screen

Fund – You can enter either ALL for all funds in your selection criteria or you can enter in a fund code or you can use the drop-down box to select from the list of funds associated with your selection criteria.

Transaction Type – You can enter either ALL for all types, or you can enter one from the drop-down list.

Start Date / End Date – This is the beginning and end date of the period that you require the report for. These dates can then be compared against the due, received or invested dates in the members investment history screen.

Report Type - The difference in the detailed report is that the contribution types are broken down and you get the date due, received and invested listed as well.

Click ‘Proceed’.

When the report has been processed and no records are found to match the selection criteria an error message is displayed:

There are no records to match the selection criteria

If records are found to match the selection criteria the report continues and options are then displayed for printing.

The options for printing are described in the document Printing From Pension Master.

Exit Program

Click ‘Exit’. This leaves the program and closes the current browser tab.
