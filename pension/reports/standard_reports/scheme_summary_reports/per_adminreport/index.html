Introduction

This program allows the user to produce a report that will report on the following areas Member Movements, Monetary Movements, Scheme Valuation, Contribution Summary and Basic Scheme Details. all held in one word document.

  • Member Movements - Lists new entrants based on date joined the scheme, risk only based on date joined company, and leavers based on date left that have moved within the date range.
  • Monetary Movement - Lists any monetary movements within the date range.
  • Scheme Valuation - Reports on the total transactions effective at the period end date.
  • Contribution Summary -Lists each contribution and its date that was received within the date range and the total of those contributions.
  • Basic Scheme Details - Details of who the report was produced by and the variables used in the report including the scheme reference numbers.

However for this report to produce data you need to update the Administration Report Configuration Administration Report Configuration the configurations of the above are set here. This is at system level however there is the same screen at scheme level if the configurations differ.

Selection Screen

Broker – Enter the Broker number or you can search from the drop-down list. This can be left blank.

Scheme – Enter the Scheme number or you can search from the drop-down list. This can be left blank but both Broker & Scheme can’t be left blank.

Start Date / End Date – This is the beginning and end date of the period that you require the report for. These dates can then be compared against the due, received or invested dates in the members investment history screen and basic member screen.

When the report has been processed and no records are found to match the selection criteria an error message is displayed:

There are no records to match the selection criteria

If records are found to match the selection criteria the report continues and options are then displayed for printing.

The options for printing are described in the document Printing From Pension Master.

Exit Program

Click ‘Exit’. This leaves the program and closes the current browser tab.
