Introduction

This program allows the user to produce a P45 for a selection of members. P45 information is extracted from the member’s records and used in the P45, along with any data entered into the P45 Details screen.

The following options and screen appear when this program is selected:

  • Run
  • Exit

Selection Screen

Run A Report

Click ‘Run’.

SELECTION CRITERIA - Select the group of members you require to produce this report for using the Scheme, Company, Category, Status, Special Code and Member Number fields (see Selection Criteria). When the selection criteria has been entered click ‘OK’ to proceed. The scheme code must be entered.

You can click ‘Cancel’ to return to the main menu and abort the P45 report.

The following screen is displayed:

P45 Details Screen

EMPLOYER PAYE REF - These fields are populated with the Tax Office Number/Reference held for the scheme in Payroll Schemes - Scheme Details screen.

Once the Employer PAYE Ref has been entered, click on ‘OK’ to proceed to the bottom part of the screen.

If there are no members found for the selection criteria entered you will get the warning

There are no records that match the selection criteria.

For each of the members selected, a check will be made to ensure that the following information is held for them. If anything is missing then a Missing Data report will be produced. If the report is produced, then it is advised that any missing data is completed to ensure the P45 is created as complete as possible.

  • NI Number
  • Sex
  • Date of Birth
  • Title, Forename and Surname
  • Current member address lines 1 to 5, and postcode
  • Scheme address lines 1 to 4, and postcode

MEMBER - This is populated with the member number. This field cannot be updated.

NAME - This is populated with the member’s fullname made up from title, forename and surname. This field cannot be updated.

LEAVING DATE - This is populated with the member’s Date Finished Pension from the Payroll Members - Pension Details screen/member’s Cessation Date from the Members - Pensioner screen.

TAX CODE - This is populated with the member’s Tax Code from the Payroll Members - Pension Details screen.

WK1/MTH1 - This is populated with the member’s Week 1/Month 1 Basis from the Payroll Members - Pension Details screen.

WEEK NUMBER - This is populated with the member’s Tax Week for the latest payment made from the Payroll Members - Current History Details screen. This field only needs to be populated when the member is paid on a cumulative basis.

MONTH NUMBER - This is populated with the member’s Tax Month for the latest payment made from the Payroll Members - Current History Details screen. This field only needs to be populated when the member is paid on a cumulative basis.

TOTAL PAY TO DATE - This is populated with the member’s Total Gross Pay - Tax Year Values To Date from the Payroll Members - Year To Date Details screen. This field only needs to be populated when the member is paid on a cumulative basis.

TOTAL TAX TO DATE - This is populated with the member’s Tax Paid - Tax Year Values To Date from the Payroll Members - Year To Date Details screen. This field only needs to be populated when the member is paid on a cumulative basis.

TOTAL PAY IN THIS EMPLOYMENT - This is populated with the member’s Total Gross Pay - Current Employment from the Payroll Members - Year To Date Details screen. This field only needs to be populated when the member is paid on a week1/month1 basis.

TOTAL TAX IN THIS EMPLOYMENT - This is populated with the member’s Tax Paid - Current Employment from the Payroll Members - Year To Date Details screen. This field only needs to be populated when the member is paid on a week1/month1 basis.

WORKS NUMBER - This is populated with the member’s Payroll Number from the Payroll Members - Personal Details screen/Members - Pensioner screen.

Details in this screen can be deleted or overwritten. If fields are blank on the screen then they will be blank in the report. All values displayed in this screen will be printed out to the P45 report. No values will be written back to Pension Master.

Once the report has been produced, a message will appear to the screen stating that the report has completed successfully.

Report successfully completed.

Click ‘OK’ to return to the report menu.

The options for printing are described in the document Printing From Payroll Master.

Exit Program

Click ‘Exit’. This leaves the program and closes the current browser tab.
