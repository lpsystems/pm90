Formulae

temp_numx

OVERVIEW

Use the temporary variable temp_numx to temporarily store numeric values within the formula.

SYNTAX

temp_numx

EXPLANATION

  • x - denotes a number e.g. temp_num1, temp_num2, temp_num3 etc.

NOTES

  1. If the temporary variable is set to a value of a non-numeric type, then an error will be generated when the formula is run.
