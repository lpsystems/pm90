Introduction

Only users set up as Supervisor status in Users may access this program.

The type of access granted can be view only or full access. For the view only option, you will not be permitted access to any menu option that allows you to add/update/delete data.

If you do not have Supervisor status, you will not be permitted access to this program.

This program allows maintenance of the Company file. This is used to:

  • Set up each company/location/paypoint for your scheme. This allows you to split up membership of a scheme into distinct sections that the scheme sponsors will recognise, and can be used, for example, to group a batch of benefit statements so as to ease distribution.
  • Record details of each companies address, contact names and addresses which can be used to generate mail merge letters to companies within your scheme.

The following options and screen appear when this program is selected:

  • Add
  • Query
  • Report
  • Letter/Cert
  • Exit

Company Details Screen

Add A Company

Click ‘Add’.

You will be asked if you wish to copy an existing Company. If you select ‘Yes’ you will be prompted to select the company code of the company you wish to copy or select one from the lookup list.

SCHEME CODE - The scheme code must exist on the scheme file. To get a lookup list of existing scheme codes click the lookup icon. When a valid scheme code has been entered the scheme description will be displayed, otherwise an error message will be displayed and the scheme code will have to be re-entered.

COMPANY CODE - Enter a code for your new Company. The combination of scheme and company code may not already exist on the company file. If a valid combination is not entered an error message will be displayed and company code will have to be re-entered.

COMPANY NAME - Enter the name of the new Company. A value must be entered in this field.

COMPANY ADDRESS - Enter the address for the Company.

POSTCODE - Enter the postcode.

COUNTRY - Enter the country.

LANGUAGE - Enter the language for the Company.

ADDITIONAL LANGUAGE - Enter the other language for the Company

TELEPHONE NUMBER - Enter the telephone number for the Company

FAX NUMBER - Enter the fax number for the Company.

E-MAIL ADDRESS - Enter the E-Mail address for the Company.

CONTACT DETAILS - Enter the name, surname and title of the main contacts at the Company. These fields are not required but can be useful as a point of reference or in a mail merge.

COMPANY ECON - (UK schemes) Enter the number for the Company

COMPANY SCON - (UK schemes) Enter the number for the Company

ACCOUNT NUMBER - Enter the account number for the Company.

TAX REFERENCE NUMBER - Enter the number for the Company

When the Company details have been entered click ‘OK’ to save the details to the company file. You can click ‘Cancel’ at any stage to exit the add option without saving the details.

Query/Search For A Company

Click ‘Query’. The following pop-up window is displayed:

Company Query Screen

You may select records by using the Scheme Code or Company Code fields. You may also use any combination (or even partial combination) of these fields in order to select the required company record(s).

If you are unsure of the code(s) to Query, click the lookup icon to see a list of codes.

Once you have entered your selection criteria, click ‘OK’ to proceed and retrieve the records from the database or click ‘Cancel’ to abort the selection and return to the Companies main menu.

Viewing Companies

To view the other records selected press N to view the next record, and P to view the previous record. If N is chosen and there are no records after the current record an error message is displayed. If P is chosen and there are no previous records an error message is displayed.

Updating Companies

Before Updating, a record must be selected through the Query option, and an error message is displayed if a record for update has not been selected.

Click ‘Update’.

The scheme and company codes cannot be changed as this is a key to the company file. When the changes have been made to the record click ‘OK’ to save the changes to file, or click ‘Cancel’ to exit the update option without saving changes.

Deleting Companies

Before Deleting, a record must be selected through the Query option, and an error message is displayed if a record for deletion has not been selected.

You will not be allowed to delete companies that contain some member records and if you try to do so, a warning message will be displayed to the screen and you will be returned to the main menu.

Click ‘Delete’.

When the delete option is selected a window will appear with the options yes or no.

If you do not wish to delete the record click ‘No’. You will return to the program menu.

If you wish to delete the record, click ‘Yes’.

A new window is displayed asking if you are sure.

If you do not wish to delete the record click ‘No’. The program returns to the main menu.

If you wish to delete the record, click ‘ Yes’. The record is deleted and the program returns to the main menu.

Broker Details

Click Menu and then select the Broker Details option and press Return.

The following screen is displayed:

Broker Details Screen

BROKER CODE - Enter a valid broker code, or click on the lookup icon for a list of broker codes. Upon selection of a valid broker code, the broker details from the Brokers program are automatically filled out on the screen and saved to the company file. If the Broker you require is not available, add it first using the Brokers program, then return to this screen to assign the broker to the company.

When all the details have been entered click ‘OK’ to save the broker details, or click ‘Cancel’ to exit entry without saving the details.

Printing Company Details

Click the ‘Report’ icon.

If records have been selected in the program before the report option was selected the following window is displayed.

Print Options Screen 1

If no records have been selected in the program before the report option was selected the following window is displayed.

Print Options Screen 2

Record Prints the current screen Record.
Query Prints the all the selected records.
Select Allows selection of records for the report using Scheme Code and Company Code fields.
Exit Closes the window and leaves the report option.

If the Select option was chosen and no records are found to match the selection criteria entered, an error message is displayed and new selection criteria should be entered.

If records are found to match the selection criteria the report continues and options are then displayed for printing.

The options for printing are described in the document Printing From Pension Master.

Instigating A Workflow Action

Click ‘Workflow’ to instigate a workflow action for the current company. A company record must be selected using the Query option before you can instigate a workflow action.

The following pop-up window is now displayed:

Workflow Instigation Popup

Enter the workflow code for the workflow action you wish to instigate or click the lookup icon for a list of workflow codes. To set up a workflow code if one to suit your purpose does not already exist, use the Workflow Management option.

Click ‘OK’ to continue or click ‘Cancel’ to abort the instigation of this workflow action.

Mail Merge Company Details

This option is used to create a mail merge file, either for a single company or a group of companies. The data output to the merge file is as follows:

  • Company Name
  • Contact Title
  • Contact Forename and Surname
  • Company Address

Click ‘Letter/Cert’.

If records have been selected in the program before the Letter/Cert option was selected the following window opens:

Letter/Cert Options Screen 1

If no records have been selected in the program before the Letter/Cert option was selected the following window opens:

Letter/Cert Options Screen 2

The menu options are as follows:

Record Prints the current screen Record.
Query Prints all the currently selected records.
Select Allows selection of records for the mail merge using the Scheme Code and Company Code fields.
Exit Closes the window and leaves the Letter/Cert option.

If the Select option was chosen and no records are found to match the selection criteria entered, an error message is displayed and a new selection criteria should be entered.

The data has now been output to the merge file and is ready to be picked up by your word processor as part of the mail merge. Please see your System Administrator for the location of the merge file.

You are now returned to the main menu.

Exit Program

Click ‘Exit’. This leaves the program and closes the current browser tab.
