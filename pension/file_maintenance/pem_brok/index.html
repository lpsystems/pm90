Introduction

Only users set up as Supervisor status in Users may access this program.

The type of access granted can be view only or full access. For the view only option, you will not be permitted access to any menu option that allows you to add/update/delete data.

If you do not have Supervisor status, you will not be permitted access to this program.

This program allows maintenance of the Broker file. This is used to:

  • Set up each broker that you deal with. This allows you then assign the appropriate broker to the scheme record in Schemes.
  • Record details of each brokers address, contact names and addresses which can be used to generate mail merge letters.

The following options and screen appear when this program is selected:

  • Add
  • Query
  • Update
  • Delete
  • Exit

Broker Details Screen

Add A Broker

Click ‘Add’.

BROKER CODE - Enter a code for your new broker record. The code entered must not already exist on the broker file. If a valid code is not entered an error message will be displayed and broker code will have to be re-entered.

BROKER NAME - Enter the name of the new broker. A value must be entered in this field and will be used in subsequent lookups for the broker details.

BROKER COMPANY - Enter the broker company using the drop-down list. If your database doesn’t use broker companies use the default already listed.

AGENT COMPANY - Enter the agent company using the drop-down list. If your database doesn’t use agent companies use the default already listed.

  • Agent
  • Introducer

BROKER ADDRESS - Enter the brokers address and postcode as appropriate.

POSTCODE - Enter the postcode.

COUNTRY - Enter the country.

BROKER TELEPHONE - Enter the telephone number of the broker.

BROKER MOBILE - Enter the mobile number of the broker.

BROKER FAX NUMBER - Enter the fax number of the broker.

BROKER E-MAIL ADDRESS - Enter the broker’s E-Mail address.

CONTACT DETAILS - Enter the name, surname and title of the main contacts of the broker. These fields are not required but can be useful as a point of reference or in a mail merge.

STATUS - Enter the status if applicable from the drop-down list:

  • Active
  • Pending
  • Closed

HAS IFA - Enter ‘Yes’ or ‘No’.

SIGNED OFF - Enter ‘Yes’ or ‘No’.

DATE OF BIRTH - Enter a date in the format ddmmyy or dd/mm/yyyy

IFA number - Enter the IFA number of the broker if applicable.

RETENTION FEE - Enter the fee if applicable.

AGENT OR INTRODUCER - Enter the Agent or Introducer:

Query/Search For A Broker

Click ‘Query’. The following pop-up window is displayed:

Broker Query Screen

You may select records by using the broker code field. You may also use any combination (or even partial combination) of these fields in order to select the required company record(s).

If you are unsure of the code(s) to Query, click the lookup icon to see a list of Broker codes.

Once you have entered your selection criteria, click ‘OK’ to proceed and retrieve the records from the database or click ‘Cancel’ to abort the selection and return to the main menu.

If any match to the selection criteria is found then the first record will be displayed to the screen and a message displayed to the bottom of the screen.

Record 1 of 2

Viewing Brokers

To view the other records selected press N to view the next record, and P to view the previous record. If N is chosen and there are no records after the current record an error message is displayed. If P is chosen and there are no previous records an error message is displayed.

Updating Brokers

Before Updating, a record must be selected through the Query option, and an error message is displayed if a record for update has not been selected.

Click ‘Update’.

The broker code cannot be changed as this is the key to the broker file. When the changes have been made to the record click ‘OK’ to save the changes to file, or click ‘Cancel’ to exit the update option without saving the changes.

Deleting Broker Details

Before Deleting, a record must be selected through the Query option, and an error message is displayed if a record for deletion has not been selected.

Also, the broker to be deleted must not be used by any scheme on your Pension Master system. If it is, a warning message will be displayed to the screen informing you of this and you will not be able to delete the broker code.

Click ‘Delete’.

You will be asked to confirm and then re-confirm deletion of this Broker.

Printing Broker Details

Click the ‘Report’ icon.

If records have been selected in the program before the report option was selected the following window is displayed.

Print Options Screen

If no records have been selected in the program before the report option was selected the following window is displayed.

Print Options Screen

Record Prints the current screen Record.
Query Prints the all the selected records.
Select Allows selection of records for the report using Broker Code field.
Exit Closes the window and leaves the report option.

If the Select option was chosen and no records are found to match the selection criteria entered, an error message is displayed and new selection criteria should be entered.

If records are found to match the selection criteria the report continues and options are then displayed for printing.

The options for printing are described in the document Printing From Pension Master.

Exit Program

Click ‘Exit’. This leaves the program and closes the current browser tab.
