Introduction

This program allows the user to produce a list of BACS data files which have been created during previous payroll runs, depending on the scheme code entered.

The following options and screen appear when this program is selected:

  • Proceed
  • Exit

BACS Data File Listing Screen

Run A Report

Click ‘Proceed’.

SCHEME CODE - Enter the scheme code required. To get a list of valid scheme codes click the lookup icon. When a valid scheme code has been entered the scheme description will be displayed.

Click ‘OK’ to proceed with the BACS data file listing. You can click ‘Cancel’ to return to the proceed menu and abort report selection.

If there are no data files held for the scheme then an error message is displayed informing the user of this. The program then returns back to the main menu.

If there are data files held for the scheme then the following screen is displayed:

BACS Data File Listing Report

Using the arrow keys or the scrollbar, select one of the data files listed in the table. Click ‘OK’ to process the report, or click ‘Cancel’ to return to the main menu.

When the selection is complete a message is displayed at the bottom of the screen:

Processing report … Please Wait

When the report has been processed and no data lines exist in the data file an error message is displayed:

The data file does not contain any data lines

The program returns to the main menu.

If data lines are found in the data file the report continues and is displayed to the screen.

The options for printing are described in the document Printing From Payroll Master.

Exit Program

Click ‘Exit’. This leaves the program and closes the current browser tab.
