Introduction

Only users set up as Supervisor status in Users may access this program.

If you do not have Supervisor status, you will not be permitted access to this program.

This program allows maintenance of Pay Element Codes used within Payroll Members . These codes can be used to add or deduct amounts from a member’s pension, during a Payroll Run.

The following options and screen appear when this program is selected.

  • Add
  • Query
  • Delete
  • Report
  • Exit

Codes Screen

Add Element Code

Click ‘Add’. This allows the user to enter the following details:

CODE - Enter the Code. The code is validated using the following rules:

  • The first character must be ‘P’ for Payroll.
  • The second character must be either ‘A’ for Addition, ‘D’ for Deduction, or ‘T’ for Tax credit.
  • The third character must not be blank.

A duplicate code cannot be added.

Characters 3 to 8 are user defineable. This will enable users to differentiate between the Codes.

DESCRIPTION - Enter a description for your Code. We recommend that you use something meaningful as this will be displayed on screens and lookup lists.

CODE TYPE - This field is set depending on the second character entered in the Code. If the second character is ‘A’, then this field defaults to ‘Addition’. If the second character is ‘D’, then this field defaults to ‘Deduction’. If the second character is ‘T’, then this field defaults to ‘Tax Credit’. This field is not accessible.

AMOUNT - Enter the amount of addition/deduction. If this field is left blank, it will default to 0.00.

TAXABLE? - Tick this field if the amount to be added or deducted is to be taxed, i.e. If the code is a ‘Deduction’ type, then tick this field if the amount is to be deducted before tax. This field does not get ticked if it is a Tax credit type code, as tax credits are not taxable.

When the Code details have been entered click ‘OK’ to save the details. You can click ‘Cancel’ at any stage to exit without saving the Code details.

Query/Search For Element Codes

Click ‘Query’.

You may select records using the Code or Description fields.

Once you have entered your selection criteria, click ‘OK’ to proceed and retrieve the records from the database or click ‘Cancel’ to abort the selection.

Click the lookup icon to see a list of existing codes.

If the selection is valid then the first record is displayed to the screen. A message is displayed to the bottom of the screen.

Viewing Element Codes

To view the other records selected click Next to view the next record, and Previous to view the previous record.

Delete Codes

Click ‘Delete’.

Before selecting delete, ensure that the appropriate code is displayed via the Query option. You are then prompted to confirm that you want to delete this code and then prompted to again confirm the deletion.

An error message is displayed if the code you are trying to delete is currently allocated to a member.

Bulk Delete Codes

Click ‘Bulk Delete’.

The Bulk Delete option can be used to delete a selection of pay elements in bulk.

Click ‘Bulk Delete’. The following screen appears:

You can select a scheme from the lookup list so only pay elements assigned to members of the scheme are looked at, or you can leave the scheme field blank and pay elements assigned to members across the database are looked at. Once the scheme field has been entered, or left blank, the bottom part of the screen will be populated with any pay elements found assigned to the selected members where the Amount YTD to 0.00:

Any pay element selected will then be deleted from member’s records. If the scheme field is left blank, then the pay element will also be deleted from the system level. If the scheme field is entered, then the pay element will also be deleted from the system level, but only if there are no other members in other schemes that are using the pay element.

Printing Element Code Details

This option allows you to print a list of one or all of the types of Element Codes available on your system.

Click the ‘Report’ icon.

The following screen will appear when this program is selected:

Report Screen

Please select one of the following options:

  • Current Pay Element Code - This will print the code currently displayed.
  • All Selected Pay Element Codes - This will print any element codes which have been queried. If no codes have been queried then the user will be informed.
  • All Pay Element Codes - This will print any codes that exist on the system.

Exit Program

Click ‘Exit’. This leaves the program and closes the current browser tab.
