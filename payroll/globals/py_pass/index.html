Introduction

This program allows the user to set, and then later change, the password that is need to be able to delete History Details from Payroll Members.

Only users set up at System Administration level will be allowed access to this program.

Setting/Changing the password

If a password hasn’t been set before, then on clicking ‘Proceed’ the user will be placed in the New Password field. Otherwise, they will be placed in the Password field, and then be required to enter a new password and confirm the new password.

Password Settings

Enter the fields as follows:

PASSWORD - Enter your password.

NEW PASSWORD - If you wish to change your password, enter a new password.

CONFIRM NEW PASSWORD - If you have entered a new password in the ‘New Password’ field, enter it again here to confirm that you have not made any typing mistakes when typing in the new password.

Once a password has been set/changed, you will be able to delete History Details from Payroll Members.

Exit Program

Click ‘Exit’. This leaves the program and closes the current browser tab.
