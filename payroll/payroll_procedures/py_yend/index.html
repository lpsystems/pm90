Introduction

This program allows the user to run a year end for a specified scheme. From this program the user can produce the P11 and P60 reports.

Only users set up as System Administrator can access this program.

If another user is running either the Backup Data program, the Restore Data program, or the Payroll Run program, then you are unable to process the Year End until the other program has completed.

The following options and screen appear when this program is selected:

  • Run
  • Exit

Tax Year End Screen

Run A Year End

Click ‘Run’.

SCHEME CODE - Enter the scheme code required. To get a list of valid scheme codes click the lookup icon. When a valid scheme code has been entered the scheme description will be displayed. This field must be entered.

ALL EMPLOYEE PAYMENTS/DEDUCTIONS HAVE BEEN PROCESSED - If the payroll run has been processed for the members then tick the box. This field must have been ticked before you can continue onto the next screen.

ALL OF YOUR PAYROLL RECORDS HAVE BEEN UPDATED - If the payroll run data has been committed for the members then tick the box. This field must have been ticked before you can continue onto the next screen.

ALL OF YOUR PAYROLL REPORT(S) HAVE BEEN PRINTED - If the payroll run payslips and payment summary report have been processed for the members then tick the box. This field must have been ticked before you can continue onto the next screen.

Click ‘OK’ to continue with the year end process, or click ‘Cancel’ to return to the tax year end menu.

The following pop-up window appears:

Proceed with Year End Window

Click ‘Yes’ to continue the year end process or click ‘No’ to return to the tax year end menu.

Printing Year End Reports

The following screen is displayed:

Tax Year End Screen

The fields are as follows:

P11 DEDUCTION WORKING SHEET (PAYE) - Click ‘Create’ to display the Pll (PAYE) report to the screen. Further details of the printing options can be found here . When you return to the screen there will be a tick displayed in the box next to the ‘Create’ button.

PRINTABLE P60 (SUBSTITUTE) - Click ‘Create’ to display the P60 (SUBSTITUE) report to the screen. A copy of the P60 report will also be saved to the ../p60_reports/printable/date/scheme/ directory. Further details of the printing options can be found here . When you return to the screen there will be a tick displayed in the box next to the ‘Create’ button.

ELECTRONIC P60 (SUBSTITUTE) - Click ‘Create’ to create a P60 (SUSBTITUE) report per member. These reports won’t be displayed to the screen. They can be found in the ../p60_reports/electronic/date/scheme/ directory where each report’s filename will contain the member number of the member the P60 is for. Once the reports have been created there will be a tick displayed in the box next to the ‘Create’ button.

Export Data For P60 Mail Merge - Click ‘Export’ to export payroll data out to a TXT file, which can then be used in the user’s own P60 template. Once the TXT file has been created there will be a tick displayed in the box next to the ‘Export’ button.

Once the user has created all necessary reports they can continue with the year end process.

Click ‘OK’ to continue, or click ‘Cancel’ to return to the tax year end menu.

The following pop-up window appears:

Year End Window

Only select ‘Yes’ if you are happy with all year end report figures. By choosing ‘Yes’, all payroll data of the tax year will be cleared out. If in doubt select ‘No’, which will return you to the tax year end menu.

Backup of Payroll Data

The Year End program automatically creates backups of the Scheme’s payroll data.

These backups are done before and after records are written to the database, and only once the user has confirmed the year end run. The two backups have descriptions of ‘Pre Year End dd/mm/yyyy’ and ‘Post Year End dd/mm/yyyy’ where dd/mm/yyyy is the Run Date.

Exit Program

Click ‘Exit’. This leaves the program and closes the current browser tab.
