Introduction

The Earlier Year Update allows the Taxable Pay and Tax values that have been submitted to the HMRC in previous tax years to be amended for specific Members. This menu option is being phased out by HMRC and is being replaced by Full Payment Submission. The table below details the tax years that the Earlier Year Update can be used:

Tax Year Earlier Year Update Full Payment Submission
2017-2018  
2018-2019
2019-2020
2020-2021  
2021-2022  

Only users set up as System Administrator can access this program.

The following options and screen appear when this program is selected:

  • Submit
  • Check
  • Exit

Earlier Year Update Submit Screen

Submit

Selecting the ‘Submit’ menu option will allow you to enter the Scheme Code and the Tax Year. Enter the Member Number, the Tax Code that was appropriate at the Tax Year, the Wk1/Mth1 setting and the adjustment for Taxable Pay and Tax values. You will need the Password that has been provided by HMRC. On clicking ‘OK’ the program will generate an XML file which will then be sent to HMRC and you will receive a confirmation message. This only indicates that HMRC have received the submission and not that the data is valid and has been accepted.

Additional information can be found on the Earlier Year Update Checklist page on the HMRC website.

Check

To check the status of the previous submission for the scheme select the ‘Check’ menu option and enter the Scheme Code. This will ask HMRC the status of the previous submission for that scheme. You will have to wait 10 seconds, a time limit imposed by HMRC, before checking your original submission.

Earlier Year Update Check Screen

The response will either be:

  • A report will be produced if errors have been found. Any errors will need to be rectified and the scheme data submitted and checked again.
  • A message saying ‘The submission has been acknowledged by the Government Gateway’ indicates that HMRC have received your ‘Check’ request and are currently processing the data submission. You will need to check later to ensure that you receive one of the two responses above.

Exit Program

Click ‘Exit’. This leaves the program and closes the current browser tab.
