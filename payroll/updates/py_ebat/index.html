Introduction

This program allows entry and maintenance of batches of pay element data for a group of members.

The data is held at the batch level until you are satisfied that it is correct, at which time you can opt to post the data to individual member records.

You can use this program to:

  • Enter batches of pay element details
  • Post data to the pay element screen in Payroll Members for each member.

The following screen is displayed when you enter this program:

Pay Elements Batch Details Screen

Add a Batch

Click ‘Add’. The following screen is displayed:

Add a Batch

SCHEME - The scheme code must exist on the scheme file. When a valid scheme code is entered the scheme description will be displayed to the screen. You can click the lookup icon to see a of scheme codes.

COMPANY - The combination of scheme and company code must exist on the company file. If a valid company code is entered the company description will be displayed to the screen. You can click the lookup icon to see a list of company codes within the chosen scheme.

CATEGORY - The combination of scheme and category code must exist on the category file. If a valid category code is entered the category description will be displayed to the screen. You can click the lookup icon to see a list of category codes for the chosen scheme and company.

STATUS - The combination of scheme and status code must exist on the status file. If a valid status code is entered the status description will be displayed to the screen. You can click the lookup icon to a list of status codes for the chosen scheme.

SPECIAL CODE - Enter a special code. If a valid special code is entered the special code description will be displayed to the screen. You can click the lookup icon to see a list of special codes.

MEMBER - Enter a member number. If a valid member number is entered the member’s name will be displayed to the screen. You can click the lookup icon to see a list of members for the chosen scheme.

Click ‘OK’ to continue, or click ‘Cancel’ to return to the main menu.

A pop-up window will be displayed to the screen which allows the user to select how the selected members should be sorted. This is defaulted to the Member Number.

The original screen will now be displayed.

The selected members and pay element codes held on the system will now be displayed on the screen. The batch number is intially set to zero and will default to the next batch number in sequence once the details have been accepted.

MEMBER - The member entered must be in the same scheme/company/category etc as that entered in the selection screen. Click the lookup icon for a list of members. Once a valid member number has been entered, the member’s name will be displayed on the screen.

CODE - The Pay Element code must exist on the system. Click the lookup icon for a list of pay element codes. Once a valid code has been entered, the pay element description will be displayed to the screen.

START - Enter the Start date the pay element code is to be effective from. If this field is left blank then it will be assumed that the pay element is active and should be processed so long as either the End date is blank or the End date is after the payroll date. If a date is entered then it must be on or before the End date, if an End date is also entered.

END - Enter the End date the pay element code stops being effective from. If this field is left blank then it will be assumed that the pay element is active and should be processed so long as either the Start date is blank or the Start date is before the payroll date. If a date is entered then it must be on or after the Start date, if a Start date is also entered.

AMOUNT - The amount is defaulted to the amount held against the pay element code. If the amount field is changed and is left blank then it will be set to 0.00.

Click ‘OK’ to save the new batch details, or click ‘Cancel’ to return to the main menu without saving any details.

Query/Search For A Batch

Click ‘Query’.

Enter details of your selection and click ‘OK’ to retrieve the records. Click ‘Cancel’ to abandon your search.

Viewing Records

To view the other records selected press N to view the next record, and P to view the previous record.

Update a Batch

Click ‘Update’.

Before updating, a record for update must be selected through the Query option.

Only the details in the lower section of the screen can be updated.

Delete a Batch

Click ‘Delete’.

The record for deletion must be selected and displayed to the screen before you will be allowed to delete it.

You will be asked if you are sure you wish to delete the record. If you answer yes the record is deleted.

The program then returns to the main menu.

Printing Batch Details

Click the ‘Report’ icon.

The options for printing are described in the document Printing From Payroll Master.

Posting Batch Details

Click ‘Post’. This option allows you to post a batch of pay element details to each members record.

A batch record must have been selected through the Query option, an error message is displayed if a record for posting has not been selected.

You will be asked if you are sure you wish to post the batch to member records.

If you have selected the ‘Yes’ option, the batch will be posted and you will be returned to the main menu.

The update will amend the relevant pay element file for each member in the batch.

When the posting has finished, you will be returned to the main menu.

Browse Batch Details

Click ‘Browse’ to view details in any screen where the data does not fit on the screen.

Once in browse mode, use the arrow keys or the scrollbar on the right of the data to navigate through the data.

Click ‘OK’ or ‘Cancel’ to leave browse mode.

Exit Program

Click ‘Exit’. This leaves the program and closes the current browser tab.
